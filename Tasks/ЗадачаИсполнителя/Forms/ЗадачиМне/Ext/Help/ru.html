Как контролировать задачи

Задачу можно поставить на контроль. При этом будет создана контрольная карточка с записями о том, что контролировать, кого и на какой срок.

Есть несколько способов работы с контролируемыми задачами:

  • Следить за всеми контрольными карточками в окне На контроле;
  • Контролировать сроки с помощью записей календаря;
  • Получать на электронную почту автоматические уведомления программы об истекающих и приближающихся контрольных сроках.

Как задать вопрос по задаче и перенести ее срок

Для этого в командной панели мини-карточки задачи предусмотрены кнопки:

 - для создания нового вопроса;

 - для отправки запроса на перенос срока.

Вопрос можно задать любому сотруднику. Запрос на перенос срока направляется автору задачи.

Если данные кнопки недоступны, скорее всего, вы забыли установить настройку программы Использовать рассмотрение вопросов выполнения задач.

Как отменить выполнение задач

Выполненную задачу можно вернуть на шаг назад только из карточки задачи. Для этого:

  1. Откройте карточку задачи. (Удобнее всего — по ссылке из панели быстрого доступа, но можно и по команде Показать выполненные меню Еще).
  2. Нажмите кнопку Отменить выполнение и, если нужно, введите комментарий.
  3. Задача вновь станет невыполненной.

Если кнопка Отменить выполнение скрыта, значит процесс перешел на следующий шаг выполнения и отменить задачу уже нельзя. В этом случае, свяжитесь с автором процесса и договоритесь об остановке или перезапуске процесса.

Советы по работе со списком

  1. С задачами удобно работать через область просмотра. При этом нет необходимости открывать карточки задач. Область просмотра включается по команде Показать область просмотра задачи в подменю Еще.
  2. В нижней части закладки Мои задачи выводится статистика о количестве задач: новых (непринятых к исполнению), просроченных, всех.
  3. По умолчанию список задач обновляется только вручную по нажатию кнопки или клавиши F5. Но в контекстном меню можно настроить Автообновление. Не рекомендуется устанавливать значения, близкие к минимальному (10 сек), т. к. это может замедлить работу программы.
  4. Список всех задач доступен по команде Все задачи из раздела Совместная работа.

Условные обозначения

 - важная задача,

Жирный шрифт - задача не принята к исполнению,

Красный шрифт - срок выполнения задачи истек,

Коричневый шрифт - у задачи есть подчиненные процессы (процессы, созданные на основании этой задачи).

 - задача делегирована (только для закладки Задачи мне),

 - ролевая задача (только для закладки Задачи мне),

 - задача выполнена с положительным/отрицательным результатом,

 - задача выполнена с замечаниями,

 - выполненные задачи, у которых нет результата выполнения (например, задачи процессов ознакомления или рассмотрения),

 - задача стоит на контроле, контрольный срок не истек/истек,

 - задача снята с контроля,

 - задача стоит на контроле у нескольких пользователей. Цвет по аналогии отражает состояние контроля у каждого из них,

 - персональные флажки. Вы видите только те флажки, которые установили сами.
