Установка флагов

Одиночные флаги удобно устанавливать и сбрасывать двойным щелчком мыши в столбце  списка писем или задач.

Для групповой установки флагов объекты нужно выделить в списке, а затем отметить флагом выбранного цвета при помощи команды , расположенной на командной панели. Групповой сброс флагов происходит аналогичным образом.

При необходимости флаг можно установить непосредственно из карточки задачи или письма.

Флаги видны только тому пользователю, который их установил.

Выбор флага по умолчанию

Флаг по умолчанию устанавливается двойным щелчком мыши в столбце .

Флаг по умолчанию можно поменять в персональных настройках программы. Эта настройка выполняется отдельно для флагов задач и писем.
