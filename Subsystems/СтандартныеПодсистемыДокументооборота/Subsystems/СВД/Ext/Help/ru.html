СВД (Система внешнего документооборота) обеспечивает автоматический обмен электронными сообщениями. Формат сообщений соответствует ГОСТу Р 53898-2010. При использовании СВД программа автоматически выполняет ряд действий:

  • выполняет обмен входящими и исходящими документами;
  • ведет учет поступающих документов в журналах;
  • связывает входящие и исходящие документы в цепочки;
  • отслеживает состояния отправленных документов.

Чтобы начать использование:

  1. Настройте серверную часть СВД.
  2. Настройте клиентскую часть СВД у каждого абонента.

Отправить документ по СВД Вы можете непосредственно из его карточки. Для этого предусмотрена команда Отправить по СВД в подменю Все действия.

Посмотреть отчеты об объеме отправки документов Вы можете как на клиенте, так и на сервере. На сервере для анализа обмена документами  используйте отчет Сообщения СВД. На стороне клиента - отчеты Отправка документов по СВД и Получение документов по СВД.
