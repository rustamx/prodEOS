Сервер СВД служит для отправки и приема сообщений пользователей СВД. Чтобы начать использовать сервер, Вы должны выполнить ряд настроек:

  1. Создать серверную информационную базу «1С:Документооборота».
  2. Опубликовать на веб-сервере созданную информационную базу и веб-сервис MEDO. Запомнить параметры публикации, т. к. они понадобятся для передачи их клиентам СВД.
  3. Установить флажок Является сервером СВД на закладке Обмен данными, появится раздел Сервер СВД.
  4. Зарегистрировать новых пользователей, которые будут подключаться к этому серверу СВД для обмена сообщениями.

Вы можете настроить взаимодействие клиента с сервером не только через веб-сервис, но и по электронной почте. Для этого:

  1. Настройте учетную запись электронной почты, через которую сервер СВД будет отправлять и загружать почту.
  2. Создайте настройку почты и укажите в ней нужную учетную запись (раздел Сервер СВД, команда Настройки почты). В дальнейшем эта настройка будет использоваться для транспорта сообщений.
  3. Создайте новый транспорт приема сообщений (раздел Сервер СВД, команда Транспорты приема сообщений). В поле Способ приема карточки транспорта укажите созданную ранее настройку почты.
  4. Создайте новый транспорт отправки сообщений (раздел Сервер СВД, команда Транспорты отправки сообщений). В поле Способ отправки карточки транспорта укажите созданную ранее настройку почты.
  5. Создайте новое правило пересылки сообщений (раздел Сервер СВД, команда Правила пересылки сообщений). Это правило сообщает серверу СВД, что сообщения для указанного пользователя необходимо перенаправлять ему по почте на адрес Email, который указан в карточке пользователя. Если для пользователя не будет указано правило пересылки, сервер СВД будет ожидать, что пользователь, которому адресован пересылаемый через сервер документ, сам обратится за ним к серверу через веб-сервис.

Вы можете использовать комбинированную схему, когда один абонент работает с сервером СВД через веб-сервис, а второй - по электронной почте.
