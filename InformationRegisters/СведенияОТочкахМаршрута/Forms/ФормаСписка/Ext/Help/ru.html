В форме выполняется управление соответствия видов работ точкам маршрута процессов.

Установленные соответствия используются в разделе учета рабочего времени.

Если в персональных настройках установлено "Добавлять работу в ежедневный отчет при выполнении задач", то при выполнении задачи, связанной с точкой маршрута, программа будет автоматически устанавливать указанный для этой точки маршрута вид работ, если для данной точки маршрута указан только один вид работ.

Если для данной точки маршрута указано несколько видов работ (задается отдельными записями), то в форме учета рабочего времени в поле вида работ программа позволяет при выборе вида использовать только те, которые заданы для этой точки маршрута.

Установка соответствия доступна сотруднику, ответственному за нормативно-справочную информацию.
