Для просмотра списка событий следует выбрать раздел "Совместная работа". В панели навигации в группе "Процессы" с помощью команды с наименованием вида процесса открыть список процессов. Затем открыть нужный процесс.

Форма списка событий открывается из формы конкретного бизнес процесса с помощью команды панели навигации "История событий задач". 

В форме показывается история изменений реквизитов задач данного процесса. Записи событий сгруппированы по задачам текущего процесса.

Для просмотра информации о конкретном событии следует открыть форму события.
