Перед вами список всех настроенных вами повторений. Управлять настройками повторений можно прямо из данного списка.

Для поиска нужной настройки используйте поле Комментарий (при создании повторения поле рекомендовано заполнять понятными описаниями).

Соответствующими кнопками командной панели вы можете:

  • Открыть карточку повторения, где можно отредактировать основные данные, например, выбрать другой шаблон процесса, используемый для повторения.
  • Изменить Расписание повторения,
  • Прекратить повторение в принудительном порядке (после этого повторение можно возобновить). Чтобы увидеть список остановленных повторения, установите одноименный флажок в нижней части данного окна.
  • Удалить повторение (повторение удаляется безвозвратно).
