Для просмотра журнала в разделе "Документы и файлы" в панели навигации выполните команду "Журнал передачи". Команда показывается, если настройка "Использовать журнал передачи" установлена.

Журнал передачи документов служит для учета в программе фактов выдачи бумажных версий документов пользователям. После возврата документа в журнале нужно поставить соответствующую отметку. 

В журнале показываются документы, переданные пользователям, в соответствии с установленным отбором (по виду передачи, виду документа, по пользователю).

Для внесения изменений факта передачи (например, о возврате) следует выполнить команду "Создать", по которой открывается форма записи о передаче документа.

На основании этих сведений можно построить отчет Документы, находящиеся у исполнителей.

В журнале передачи документов можно указать, на какой срок передается документ. Для этого в записи журнала передачи требуется заполнить реквизит «Срок возврата».
В журнале передачи можно отразить передачу документа не только собственным сотрудникам, но и контактному лицу контрагента.

Журнал передачи по конктреному документу можно открыть с помощью команды панели навигации "Журнал передачи" формы документа. Команда показывается, если настройка "Использовать журнал передачи" установлена.
