Форма "Запись журнала передачи" документа служит для учета в программе фактов выдачи бумажных версий документа пользователям. После возврата документа в карточке журнала нужно поставить соответствующую отметку "Возвращен". 

Для внесения записи о факте передачи в журнале передачи всех документов или в журнале передачи документа выполните команду "Создать". Для внесения записи о передачи из карточки или списка документов выполните команду "Создать на основании \ Запись журнала передачи". Укажите тип передаваемого экземпляра (оригинал или копия), укажите номер экземпляра. Этот номер не может быть больше, чем количество экземпляров, указанное в карточке документа (выполняется проверка при сохранении данных о передаче). Укажите пользователя или контактное лицо, которому передается документ. Для дополнительного описания можно ввести текст в поле "Комментарий".

Для учета факта возврата документа следует открыть карточку записи передачи документа и поставить отметку "Возвращен". При этом в поле даты возврата заносится текущаяя дата, значение которой можно исправить.
