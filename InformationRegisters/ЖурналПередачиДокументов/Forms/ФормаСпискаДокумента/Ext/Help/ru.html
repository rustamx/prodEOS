Для открытия журнала передачи документа следует в форме документа (входящего, исходящего или внутреннего) выполнить команду панели навигации "Журнал передачи". 

Журнал передачи документов служит для учета в программе фактов выдачи бумажных версий документов пользователям. После возврата документа в журнале нужно поставить соответствующую отметку. 

Для внесения изменений факта передачи (например, о возврате) следует выполнить команду "Создать", по которой открывается форма записи о передаче документа.

На основании этих сведений можно построить отчет Документы, находящиеся у исполнителей.

Журнал передачи по всем документам можно открыть с помощью команды панели навигации "Журнал передачи" раздела "Документы и файлы".
