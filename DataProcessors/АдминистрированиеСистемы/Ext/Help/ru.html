Выполнение функций по администрирование программы разрешено только пользователю с ролью "Полные права" (см. раздел "Особенности справочника "Профили групп доступа""):

  • Удаление помеченных объектов
  • Управление полнотекстовым поиском
  • Блокировка соединений с информационной базой
  • Консоль регламентных заданий
  • Активные пользователи
  • Управление настройками форм
  • Журнал регистрации
  • Контроль журнала регистрации
  • Очистка истории работы всех пользователей
