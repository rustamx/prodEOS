Заполните обязательные для заполнения поля внутреннего документа. В списке можно отметить файлы для импорта в документ. Если в почтовом сообщении есть файлы с подпиями (ЭП), то можно установить соответствие между файлом и подписью с помощью перетаскивания подписи на подписываемый файл. При этом будет произведена проверка подписи. Если проверить подпись не удалось, то она будет отображаться в списке красным цветом. Файлы подписи (ЭП) отображаются в списке курсивом.
