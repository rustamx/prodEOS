Входящий, исходящий и внутренний документ или файл можно отправить по электронной почте при помощи команды  в командной панели списка или карточки документов и файлов.

При этом в текст письма автоматически вставляется ссылка на исходный документ или файл, которую удобно использовать в дальнейшей переписке для перехода к нужным данным.

В карточке письма укажите:

  • получателя письма (устанавливается автоматически при отправке входящих или исходящих документов, если у соответствующих контрагентов или их контактных лиц указан адрес электронной почты);
  • получателей копий письма
  • скрытых получателей копий
  • тему письма;
  • файлы, которые нужно отправить.

При отправке по почте можно установить флажок Шифровать отправляемые файлы. В этом случае необходимо указать указать список сертификатов, для которых будет шифроваться файл по кнопке Настроить шифрование.

Примечание
Зашифрованные файлы будут иметь расширение, указанное в персональных настройках ЭП. Если в настройках расширение не задано, то зашифрованные файлы будут иметь расширение "p7m".

 

Если в настройках отправки было указано "Прикладывать файл взаимодействия СЭД по ГОСТ Р 53898-2010", то к отправляемому письму автоматически будет приложен файл "1C-53898-2010.XML". В этом файле содержится описание документа, соответствующее ГОСТу Р 53898-2010 "Системы электронного документооборота. Взаимодействие систем управления документами. Требования к электронному сообщению". Эта настройка доступна в разделе Настройкаи администрировани - Персональные настройки - Работа с почтой - Прикладывать файл взаимодействия СЭД.
