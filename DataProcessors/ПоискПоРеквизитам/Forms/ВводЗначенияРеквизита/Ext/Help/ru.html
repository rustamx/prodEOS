Форма ввода списка значений предназначена для указания списка значений, которые может принимать реквизит.

В левой части формы отображается список всех доступных значений реквизита. В правой части располагается список выбранных значений.

Для выбора одного или нескольких значений из списка доступных необходимо выделить одно или несколько значений в списке доступных значений и выполнить двойной щелчок мышью либо нажать на копку ">", расположенную между списками.

Для удаления значений из списка выбранных необходимо выделить значения и либо выполнить двойной щелчок мышью, либо нажать на кнопку "<", расположенную между списками.

Для того, чтобы указать, что необходимо найти документы и файлы с незаполненным реквизитом, необходимо поставить отметку в поле "Не заполнено".
