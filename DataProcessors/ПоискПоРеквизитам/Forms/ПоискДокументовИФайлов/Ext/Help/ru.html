Общая информация

Форма поиска документов и файлов по реквизитам открывается командой "Поиск по реквизитам" на панели навигации Рабочего стола и раздела "Документы и файлы", а также по сочетанию клавиш Ctrl+Shift+S в любом из разделов.

Форма поиска документов и файлов по реквизитам может отображаться в двух режимах - в режиме задания условий и в режиме отображения результатов поиска.

Для того, чтобы найти документы или файлы по их реквизитам, необходимо выполнить следующие действия:

  • Перейти в режим задания условий (открывается по умолчанию при открытии формы)

  • Выбрать область поиска. Это могут быть:

Область поиска  Описание
Внутренние, входящие и исходящие документы Поиск будет выполняться среди документов всех типов.
Внутренние документы Поиск будет вестись среди внутренних документов
Входящие документы Поиск будет вестись среди входящих документов
Исходящие документы Поиск будет вестись среди исходящих документов
Файлы

Поиск будет вестись среди файлов

Расширенный поиск

Пользователь указывает, среди каких объектов ведется поиск. Пользователь может задавать условия поиска произвольной сложности с различными логическими объединениями.

 

  • Указать значения реквизитов, по которым необходимо выполнить поиск. Для этого необходимо в колонке "Что искать" в соответствующей строке нажать на кнопку с иконкой . В результате откроется форма, в которой необходимо указать значение реквизита. Внешний вид формы для ввода значения реквизита зависит от типа реквизита.

  • После указания значения всех реквизитов, по которым необходимо выполнить поиск, необходимо нажать на кнопку "Искать"

 

В результате выполнения поиска форма переключится в режим показа результатов поиска, в котором становятся доступными списки найденных документов и файлов.

Для перехода обратно в режим задания условий поиска необходимо нажать на кнопку "Изменить условия" в режиме показа результатов поиска. Для того, чтобы обновить результаты поиска по заданным ранее условиям, необходимо нажать на кнопку "Повторить поиск".

 

Работа с шаблонами поиска

Шаблоны поиска позволяют пользователю сохранять условия поиска с тем, чтобы в будущем он имел возможность их повторного использования.

Список шаблонов отображается на форме поиска по реквизитам, когда она находится в режиме задания условий. Элемент списка "Новый поиск" позволяет очистить введенные условия поиска, не изменяя область поиска.

Пользователь может выполнить следующие команды:

Команда Описание
 Переименовать Позволяет переименовать шаблон поиска
 Сохранить как

Сохраняет шаблон поиска. После выполнения этой команды шаблон в списке отображается с иконкой

 Удалить

Удаляет выделенный шаблон

Текущие условия поиска можно сохранить в качестве шаблона, нажав на кнопку "Сохранить шаблон..." в режиме задания условий поиска. Кнопка расположена в командной панели списка шаблонов поиска.

При выходе из программы все шаблоны поиска без иконки  (временные шаблоны) удаляются.
