Если на сервере 1С:Предприятия не установлена программа распознавания (CuneiForm), например в случае если сервер работает под Линукс, на одном из клиентских компьютеров, работающих под управлением ОС Microsoft Windows нужно запустить тонкий или толстый клиент, и в нем открыть обработку «Автоматическое распознавание изображений». Эта обработка в автоматическом режиме будет распознавать файлы изображений. По умолчанию интервал времени выполнения равен 60 секундам.

После открытия обработки нужно нажать кнопку "Старт", чтобы в автоматическом режиме распознавались файлы изображений (порциями, по умолчанию по 100 файлов).
Чтобы распознать сразу все нераспознанные файлы, надо нажать кнопку "Распознать все".
