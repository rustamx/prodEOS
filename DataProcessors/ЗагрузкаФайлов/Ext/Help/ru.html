С помощью этой обработки можно выполнить автоматическую загрузку файлов из каталога на жестком или сетевом диске в папку файлов информационной базы. При этом выполняется автоматическая загрузка всех подкаталогов.

Загрузка файлов выполняется с учетом подкаталогов, для каждого из которых создается своя подпапка в информационной базе. При загрузке выполняется проверка имен файлов и если в папке уже есть файл с таким же именем как загружаемый, то загрузка этого файла не выполняется.

Примечание:
После успешной загрузки в информационную базу файлы будут удалены с диска.
