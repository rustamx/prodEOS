Электронная почта — технология и предоставляемые ею услуги по пересылке и получению электронных сообщений (называемых «письма» или «электронные письма») по распределённой (в том числе глобальной) компьютерной сети. (Википедия)

Почтовый ящик — место в глобальной сети, предназначенное для хранения писем. Каждый почтовый ящик имеет адрес вида имя_пользователя@имя_домена.

Почтовые ящики находятся на почтовых серверах. Почтовый сервер — это набор технических и программных средств, позволяющий обслуживать прием, хранение и отправку почтовых сообщений.

Пользователи работают с письмами с помощью почтовых программ (почтовых клиентов) — специальных программ, установленных на компьютере и позволяющих читать письма в почтовом ящике, создавать и отправлять письма в другие почтовые ящики.

Почтовый профиль — набор параметров, используемый почтовой программой для подключения к почтовому ящику пользователя.

В системе 1С:Документооборот 8 можно работать как с почтовыми программами, установленными на компьютере пользователя, так и непосредственно с почтовыми ящиками.

Работа с почтовым ящиком (без запуска почтовой программы) может потребоваться для отправки почты через общий корпоративный почтовый ящик или для получения почты из общего почтового ящика.

Работа с почтовой программой, установленной на компьютере пользователя, может потребоваться при необходимости загрузить письмо, присланное на личный почтовый ящик или для отправки документов и файлов через личный почтовый ящик.

Для работы с почтовым ящиком необходимо настроить профиль с видом SMTP/POP3. Для этого:

  • Добавьте профиль с типом SMTP/POP3 с помощью команды меню Добавить - SMTP/POP3.
  • Заполните карточку учетной записи электронной почты. Параметры настройки можно уточнить у системного администратора.

 

Для работы с почтовой программой, поддерживающей протокол MAPI необходимо настроить профиль MAPI. Почтовые программы, поддерживающие протокол MAPI: Microsoft Outlook, The Bat, Mozilla Thunderbird и другие. Поддерживает почтовая программа протокол MAPI или нет можно уточнить у системного администратора.

  • Добавьте профиль с типом MAPI с помощью команды меню Добавить - MAPI.
  • Заполните карточку профиля MAPI для загрузки или для отправки.

 

Если на компьютере установлена программа Microsoft Outlook, то для работы с ней необходимо настроить профиль Microsoft Outlook. Для этого:

  • Добавьте профиль с типом Microsoft Outlook с помощью команды меню Добавить - Microsoft Outlook.
  • Заполните карточку профиля Microsoft Outlook для загрузки или для отправки.

 

Если на компьютере установлена программа The Bat!, то для работы с ней необходимо настроить профиль The Bat. Для этого:

  • Добавьте профиль с типом The Bat с помощью команды меню Добавить - The Bat.
  • Заполните карточку профиля The Bat.

Профиль The Bat можно использовать только для отправки писем.

 

Если на компьютере установлена программа Mozilla Thunderbird, то для работы с ней необходимо настроить профиль Mozilla Thunderbird. Для этого:

  • Добавьте профиль с типом Mozilla Thunderbird с помощью команды меню Добавить - Mozilla Thunderbird.
  • Заполните карточку профиля Mozilla Thunderbird.

Профиль Mozilla Thunderbird можно использовать только для отправки писем.

 

Для того чтоб указать, что профиль должен использоваться по умолчанию выделите его и нажмите кнопку По умолчанию в командной панели списка. Если не указан профиль по умолчанию для отправки, то при отправке файлов будет выведен диалог выбора профиля. Профиль по умолчанию для загрузки используется при загрузке электронной почты.

Для профиля можно указать необходимость отображения его в списке выбора. Если для профиля снят флаг Использовать, то он не будет отображаться в соответствующем списке выбора профилей при отправке или при загрузке сообщений.

Форма настройки почтовых профилей может быть открыта:

  • из формы Персональные настройки с помощью кнопки Настроить профили на закладке Работа с почтой;
  • из формы выбора профилей при отправке сообщения;
  • из формы выбора профилей при загрузке почтовых сообщений.
