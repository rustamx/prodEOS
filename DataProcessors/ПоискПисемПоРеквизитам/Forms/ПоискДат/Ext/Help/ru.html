Установите период, за который были получены письма:

  • установите значение только в поле С, чтобы искать письма с определенной даты по текущую.
  • установите значение только в поле По, чтобы искать письма только по определенную дату.
  • установите значения в обоих полях, чтобы задать период для поиска.
