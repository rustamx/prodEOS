Виды внутренних документов настраиваются ответственным за НСИ в соответствии с правилами делопроизводства и документооборота, установленными в организации.

Вид определяет правила формирования регистрационного номера документов, нормативный срок их исполнения и набор  дополнительных реквизитов и свойств.

Для настройки видов в разделе "Нормативно-справочная информация" в панели навигации выполните команду Виды документов, в открывшейся форме выберите "Внутренние документы". В списке видов документов создайте перечень нужных видов (например, "Бухгалтерский отчет", "Договор", "Приказ по кадрам" и т.д.).

На закладке Настройки выполняется следующая настройка:

  • Использовать срок исполнения - для некоторых видов документов не требуется учитывать срок исполнения. С помощью данной настройки учет срока исполнения может быть отключен. Если снят, то в карточке документа данного вида станет невидимым поле Срок  исполнения;
  • Вести учет по номенклатуре дел - если установлен, то в карточке документа данного вида появится реквизит Номенклатура дел.
    В нумераторы включена возможность использовать индекс по номенклатуре дел в регистрационном номере документа. Индекс по номенклатуре дел выбирается из реквизита Номенклатура дел карточки документа данного вида, поэтому для использования индекса по номенклатуре дел должна быть включена данная настройка.
  • Необходима печать штрихкода при создании документа - если установлен, то при закрытии карточки только что созданного внутреннего документа данного типа будет осуществляться проверка того, был ли напечатан штрихкод созданного документа (любым из возможных вариантов). Если штрихкод не был напечатан, то система выдаст предупреждение с запросом подтвердить закрытие формы.
  • Вести учет по контрагентам - если установлен, то в карточке документа данного вида будут доступны поля учета данных о контрагенте, контактном лице контрагента и подписавшем со стороны контрагента;
  • Учитывать срок действия - если установлен то в карточке документа данного вида появятся поля для указания срока действия документа и порядка продления документа.
  • Учитывать сумму документа - если установлен, то для документов данного вида можно будет указать сумму и валюту документа.
  • Учитывать недействующие документы - если установлен, то документы данного вида могут быть отменены или расторгнуты. При установке в карточке документа данного вида появляется флаг Не действует и ссылка на документ, который делает его не действующим;
  • Является договором - данный признак означает, что документы с этим видом являются договорами. При установке данного признака автоматически устанавливаются признаки Вести учет по контрагентам, Учитывать срок действия, Учитывать сумму документа, Учитывать недействующие документы и сбрасывается признак Является комплектом документов.
  • Является комплектом документов - данный признак означает, что документы с этим видом являются комплектами других документов и файлов. Установить данный признак можно при условии что не установлен ни один из признаков: Является договором, Вести учет по номенклатуре дел, Учитывать срок действия, Учитывать недействующие документы.
  • Автоматически вести состав участников рабочей группы - для документов данного вида в рабочую группу будут автоматически добавляться новые участники рабочих процессов по документу.
  • Для документов этого вида заполнение рабочей группы является обязательным - для документа данного вида контролируется заполнение рабочей группы.
  • Подписывать резолюции электронной подписью - если флажок установлен, то для документов данного вида утверждение каждой резолюции ЭП будет обязательным.


На закладке Нумерация настраиваются правила нумерации документов данного вида.

Документы можно нумеровать автоматически или вручную.

При автоматической нумерации регистрируемый документ автоматически получит номер заданного формата. При нумерации вручную номер документу присваивает пользователь в ходе регистрации.

Перед созданием правила автоматической нумерации нужно настроить нумератор и формат номера, которые будет использовать правило.

Для одного вида документов можно использовать разные нумераторы

Для одного вида документа можно задавать разные правила. Это необходимо в тех случаях, когда требуется отдельная нумерация для документов, оформляемых на разные организации, в рамках разных проектов и т. п.

При автоматической нумерации в столбце Нумеровать указывается вид нумерации, используемый нумератор и формат нумера. Например: Автоматически, Входящие документы, 12345-АКТ.

В каждой строке можно указать, как нумеровать документы - вручную или автоматически, какой использовать нумератор для автоматической нумерации, а также набор измерений, для которых действует указанная настройка нумерации. Например, для вида документа Служебная записка можно настроить один нумератор для подразделения Отдел кадров и другой для Бухгалтерии. Каждый из них может иметь свой формат номера и периодичность нумерации.
