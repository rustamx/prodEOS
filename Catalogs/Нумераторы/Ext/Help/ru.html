Справочник "Нумераторы" служит для настройки правил формирования регистрационных номеров у входящих, исходящих и внутренних документов.

Для создания и редактирования нумераторов в разделе "Нормативно-справочная информация" в панели навигации выполните команду "Нумераторы", в списке с помощью команд "Создать" и "Изменить" создавайте и редактируйте требуемые нумераторы.

Нумератор назначается видам документов. Один нумератор можно назначить сразу нескольким видам, например, для того чтобы поддерживать для них сквозную нумерацию.

Наименование нумератора следует выбирать по области использования. Например, для вида документа "Трудовые договоры" можно создать одноименный нумератор "Трудовые договоры".

В поле "Периодичность" указывается периодичность использования данного нумератора. Для каждого нового периода нумератор начинается с начала.

Если правила регистрации документов предусматривают ведение отдельной нумерации документов разных организаций, то в нумераторе нужно установить флажок "Независимая нумерация по организациям". В этом случае документы того вида, которому назначен этот нумератор, будут иметь регистрационный номер одного формата, но с индивидуальной нумерацией по каждой организации.

Каждый нумератор имеет свой собственный нарастающий регистрационный номер, который не зависит от других нумераторов.

Нумератор определяет формат регистрационного номера, который может сконструировать (собрать из готовых фрагментов) в карточке нумератора. Формат регистрационного номера может включать в себя различные служебные поля, отмеченные квадратными скобками и разделители (например, тире, косая черта).

Если разделитель помещен внутрь квадратных скобок, внутри которых указано служебное поле, то он будет добавлен к номеру только в том случае, если будет добавлено служебное поле. Например, если  формат регистрационного номера имеет вид [Номер][-ИндексОрг], то тире будет подставлено в номер только в том случае, если в списке индексов нумерации указан индекс соответствующей организации.

Нумератор позволяет формировать номер документа с использованием в качестве префикса или суффикса номера связанного документа. Для этого используется служебное поле [НомерСвязДок], которое можно добавить в формат номера. При этом становится доступным поле "Тип связи", в котором требуется указать тип связи, определяющий связанный документ, из которого будет взят номер.

Нумератор позволяет вести раздельную нумерацию в разрезе связанного документа. Для этого в карточке нумератора необходимо установить флаг "Независимая нумерация по связанному документу" и указать тип связи, определяющий связанный документ. Например, если такой нумератор присвоен виду документа «Доп. соглашение», то доп. соглашения, относящиеся к одному договору, будут иметь возрастающую нумерацию, начиная с 1, которая будет независима от нумерации доп. соглашений, относящихся к другому договору.

Посмотреть результат сформированного нумератора можно в поле "Пример" (внизу формы).

Список текущих номеров (в том числе для каждой организации) можно посмотреть при помощи команды "Все номера" панели навигации карточки нумератора.

Список видов документов, для которых назначен нумератор можно посмотреть при помощи команды "Кому назначен" панели навигации карточки нумератора.
