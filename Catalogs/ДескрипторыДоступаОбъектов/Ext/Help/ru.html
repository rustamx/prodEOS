Что такое дескриптор доступа

Дескриптор доступа презназначен для хранения информации о правах объектов программы (документов, папкок, процессов и т.п.). Эту информацию можно использовать для детального анализа прав данных объектов.

Из карточки дескриптора можно получить следующую информацию:

  • все ключевые настройки (организация, гриф доступа, вид документа, папка и другие), по которым рассчитаны права доступа к выбранному вами объекту;
  • данные о том, для каких объектов используется данный дескриптор;
  • права, рассчитанные по дескриптору.

Дескрипторы создаются не для каждого объекта программы, а для комбинации ключевых настроек. Например, для сотни документов с одинаковым видом документа, грифом доступа и т.д. будет использован один дескриптор.

Как пользоваться дескриптором 

Все данные, которые использовались для расчета прав, отображаются ссылками на закладке Дескриптор. 

Чтобы отличать похожие дескрипторы друг от друга, используйте данные полей Хеш-сумма и Уникальный идентификатор. Также эти данные помогут определить смену дескриптора доступа того или иного объекта.  

Чтобы перессчитать права по данному дескриптору, нажмите Обновить на закладке Права доступа. Перерасчет выполнится немедленно, даже если права на объект стояли в очереди отложенного обновления прав.

Список объектов, которые используют данный дескриптор, расположен на закладке Объекты.
