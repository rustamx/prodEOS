Справочник "Контрагенты" служит для учета юридических и физических лиц, с которыми Ваша организация обменивается документами. Сведения из этого справочника используются, например, при регистрации входящих и исходящих документов.

Просмотр списка контрагентов выполняется в разделе "Документы и файлы". В панели навигации следует выполнить команду "Контрагенты". На экран выводится список контрагентов. Управление сведениями (добавление нового, копирование, редактирование имеющегося, установка/снятие пометки удаления, поиск) выполняется с помощью команд списка.

В карточке контрагента сведения о контрагенте учитываются на двух закладках. На закладке "Основные сведения" учитываются основные описательные данные, на закладке "Адреса, телефоны" - ведется список адресов и телефонов контрагента. В карточке указывается вид контрагента - "Юридическое лицо", "Физическое лицо", "Индивидуальный предприниматель", "Юридическое лицо за пределами РФ". Указанный вид контрагента определяет состав полей карточки, набор дополнительных реквизитов, правила корректного заполнения ИНН, КПП, ОКПО.

Кроме того, с помощью команд панели навигации, размещенных в карточке контрагента в группе "Перейти", осуществляется управление связанной с контрагентом информацией: учет контактных лиц, просмотр переписки с контрагентом, внутренние документы, в которых фигурирует контрагент, просмотр дел (томов), в которых хранятся документы по этому контрагенту.

При создании контрагента проверяется его уникальность по реквизитам "ИНН", "КПП", "Наименование" и "Полное наименование". Если в справочнике уже есть записи, у которых значение хотя бы одного из указанных реквизитов совпадает со значением соответствующего реквизита создаваемого контрагента, и все эти записи доступны текущему пользователю по правам доступа, то откроется список этих объектов. Если среди найденных контрагентов с дублирующими значениями реквизитов есть хотя бы одна запись, недоступная текущему пользователю по правам доступа, то вместо списка найденных контрагентов система выдаст сообщение "У пользователя недостаточно прав на исполнение операции над базой данных", и процесс сохранения нового контрагента прервется.

Контрагента можно поставить на контроль:

  • следить за сроком и ходом выполнения работ по контрагенту,
  • переносить контрольный срок,
  • контролировать отдельных пользователей,
  • снимать контрагента с контроля,
  • выводить аналитические отчеты,
  • смотреть, каких контрагентов контролируют ваши коллеги.

Программа сама отследит истекающие и приближающиеся контрольные сроки и сообщит вам об этом по почте.

Чтобы поставить контрагента на контроль, нажмите в командной панели карточки или списка контрагентов. При этом откроется Контрольная карточка, в которой нужно установить срок, указать контролируемое действие и исполнителя. Если контрагент уже контролировался ранее, кнопка откроет список всех контрольных карточек когда-либо созданных по контрагенту. В списке можно посмотреть, кто сейчас контролирует контрагента, открыть контрольные карточки и, при наличии прав, изменить параметры контроля.

Состояние контролируемого контрагента можно посмотреть в списке (колонка Контроль): - контрагент на контроле, контрольный срок не истек; - контрагент на контроле, контрольный срок истек; - контрагент снят с контроля. Значки с двумя означают, что контрагент находится на контроле у нескольких пользователей. Цвет по аналогии отражает состояние контроля у каждого из них.

После постановки на контроль рекомендуется настроить уведомления об истекающих или пропущенных сроках (в настройках программы) и следить за всеми контролируемыми документами в окне На контроле.
