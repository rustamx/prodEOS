Как выбрать формат сообщения

Для каждого транспорта Вы должны определите формат, в который будут записываться сообщения при отправке. В поле Формат сообщения доступны следующие варианты выбора: 

  • ГОСТ Р 53898-2010 - сообщение записывается согласно требованиям ГОСТа;
  • Сообщение 1С:Документооборот - сообщение также записывается согласно ГОСТ Р 53898-2010, но с возможностью добавления ЭП.

Как настроить способ доставки сообщений через веб -сервис

Чтобы настроить прием и отправку сообщений с помощью веб-сервиса 1С:Документооборота, выполните следующие действия:

  1. Нажмите на кнопку Выбрать, в новом окне откроется список доступных способов отправки.
  2. Двойным щелчком мыши выберите Веб-сервис 1С:Документооборота из списка.
  3. Заполните все необходимые поля в настройке доставки сообщений через веб-сервис.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.

Более подробную информацию о настройке веб-серверов для работы с "1С:Предприятием" читайте в книге "1С:Предприятие 8.2. Руководство админстратора".

Как настроить способ доставки сообщений через email

Чтобы настроить прием и отправку сообщений через email в формате 1С:Документооборота, выполните следующие действия:

  1. Нажмите на кнопку Выбрать, в новом окне откроется список доступных способов отправки.
  2. Двойным щелчком мыши выберите Email в формате 1С:Документооборота из списка.
  3. Заполните все необходимые поля в настройке доставки сообщений по email.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.

Как заполнить список участников СВД

Участники СВД - это организации или государственные учреждения, которые будут обмениваться сообщениями через сервер СВД по этому транспорту.

Чтобы заполнить список участников, выполните следующие действия:

  1. Нажмите на кнопку Добавить, станет активным поле выбора участника.
  2. Выберите контрагента из списка или введите с клавиатуры вручную.
  3. Укажите уникальное имя участника на сервере СВД в колонке Имя участника.

Внесите в список участников СВД с помощью этого транспорта свою организацию и ее уникальное имя на сервере.
