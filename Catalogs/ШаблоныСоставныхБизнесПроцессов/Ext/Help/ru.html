Для создания и редактирования составных шаблонов процессов в разделе "Нормативно-справочная информация" в панели навигации следует выполнить команду "Шаблоны составных процессов". 

Шаблоны составных процессов имеют две предопределенные группы: "Обработка входящего документа" и "Обработка исходящего документа". Также при доработке типовой конфигурации состав групп может быть расширен другими видами. 

При выборе одной из групп шаблоны данного вида будут показаны в списке. В списке с помощью команд "Создать" и "Изменить" создавайте и редактируйте шаблоны.

Составной шаблон состоит из цепочки простых шаблонов. В левой части карточки шаблона показывается представление действия, для которого нужно указать простой шаблон. Для "Обработка входящего документа" - это "Рассмотрение" + "Исполнение\Ознакомление" + "Списание в дело". Для этих действий можно выбрать шаблоны следующих видов: Рассмотрение, Исполнение или Ознакомление и Поручение (списать в дело). Для "Обработка исходящего документа" - это "Регистрация", "Согласование" и "Утверждение". Для этих действий можно выбрать шаблоны следующих видов: Регистрация, Согласование и Утверждение.

В шаблоне процесса можно указать различные данные, которые будут использованы при создании нового процесса на основании документа или при ручном заполнении нового процесса из шаблона (команда "Заполнить по шаблону" меню "Все действия" в карточке процесса):

  • Наименование шаблона;
  • Тип шаблона - выбираются из перечня имеющися типов составных шаблонов (в типовой конфигурации это "Обработка входящего документа" и "Обработка исходящего документа"). При выборе типа шаблона таблица "Реквизиты" заполняется необходимыми реквизитами (типами шаблонов), описанными в выбранном составном шаблоне. Для дальнейшей настройки следует в каждой строке (реквизите) в колонке "Шаблон вложенного процесса" указать конкретный шаблон;
  • Ответственный - выбирается из справочника пользователей;
  • Комментарий - краткое описание шаблона;
  • Наименование - наименование процесса, создаваемого по данному шаблону.

Поля карточки процесса, заполненные из шаблона, становятся недоступными для ручного изменения пользователем.

Поля "Наименование шаблона", "Ответственный" и "Комментарий" носят информационно-справочный характер и не влияют на создание новых процессов по этому шаблону.

Примечание: права на составные процессы складываются из прав всех подчиненных им процессов, в том числе и с учетом переадресации.

Рабочая группа шаблона - это список пользователей и ролей, которые могут пользоваться этим шаблоном. Если рабочая группа не заполнена, то он доступен всем. При заполнении обязательно внесите текущего пользователя, иначе программа не даст записать шаблон. Изменять шаблон может только пользователь, который указан в поле Ответственный.
