Разделы форума служат для упорядочивания данных и ограничения прав доступа к ним.

В поле "Наименование" задается наименование раздела в виде произвольной сроки.

В поле "В папке" выбирается раздел, в котором будет размещен создаваемый раздел.

В поле "Описание" вводится текст описания использования раздела (произвольный текст).

За разделом может быть назначен ответственный (по умолчанию заполняется данными текущего пользователя).

Существует предопределенный раздел "Предметные темы". Данный раздел предназначен для тем форума, связанных с каким-либо предметом, например, внутренним документом. Такие темы могут располагаться только в указанном разделе. В разделе "Предметные темы" запрещено создавать подразделы и темы форума без предмета.

После записи данных о разделе становится доступно задание прав доступа в разделе (распространяется на все общие темы форума, размещенные в разделе). Для этого следует выполнить команду "Настроить права". На экран выводится форма для настройки доступа. Права доступа настраиваются администратором программы.

Права для тем связанных с предметом определяются исходя из прав предмета. Если предмет доступен пользователю для чтения, то он может читать и оставлять новые сообщения в теме форума, связанной с данным предметом. В случае, когда права на чтение предмета отсутствуют, то тема форума, связанная с данным предметом, тоже недоступна. На права предметных тем дополнительно влияют настройки прав предопределенного раздела "Предметные темы".

  • "Чтение" - данная настройка влияет на видимость раздела "Предметные темы" в списке разделов. При отключении данного права доступ к предметным темам всё равно остаётся, к ним можно перейти по навигационной ссылке или через команду "Форум" в панели навигации предмета темы.
  • "Добавление" - данные настройки не влияют на доступ к предметным темам;
  • "Изменение" - данная настройка даёт право изменения сообщений других пользователей, если есть доступ к предмету темы форума;
  • "Удаление" - данная настройка даёт право удаления сообщений других пользователей, если есть доступ к предмету темы форума;
  • "Изменение разделов" - данная настройка даёт право добавления и изменения разделов;
  • "Управление правами" - данная настройка даёт право изменять права раздела.

 

Для тем без предмета, права папки влияют следующим образом:

  • "Чтение" - данное настройка влияет на право чтения раздела и тем в нём;
  • "Добавление" -  данная настройка влияет на право добавления тем и сообщений в раздел;
  • "Изменение" - данная настройка даёт право изменения сообщений других пользователей в разделе;
  • "Удаление" - данная настройка даёт право удаления сообщений других пользователей в разделе;
  • "Изменение разделов" - данная настройка даёт право добавления и изменения разделов;
  • "Управление правами" - данная настройка даёт право изменять права раздела.
