Как добавить визу от лица другого сотрудника

В программе можно добавить визу за другое согласующее лицо. Например, секретарь добавляет визу от лица руководителя. Для этого достаточно выбрать (или ввести вручную) руководителя в поле Согласующее лицо. При этом секретарь будет отмечен как Автор визы.

В зависимости от правил, установленных на предприятии, руоводитель может доверить секретарю вносить только текст визы (результат согласования проставить собственноручно) или полностью передать право добавления виз.

Как подписать визу электронной подписью

Для подписания визы и проверки ЭП используется группа команд ЭП на панели действий карточки визы.

При этом виза становится недоступна для редактирования. Изменить визу можно только после удаления всех ее ЭП.

В зависимости от правил документооборота на предприятии электронная подпись визы может считаться либо обязательной, либо нет. Настроить обязательное подписание визы ЭП можно из карточки процесса Согласование. Для этого при создании процесса нужно установить флажок Подписывать визы ЭП при согласовании. Обратите внимание, что после установки этого флажка программа не позволит сохранять неподписанные визы.

Что такое визы согласования

Визы - это результаты согласования документов. Они хранятся на закладке Визы карточки документа, где можно ознакомиться с историей согласования и распечатать лист согласования.
