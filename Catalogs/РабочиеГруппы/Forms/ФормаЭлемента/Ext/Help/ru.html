Рабочая группа

Просмотр и редактирование свойств рабочей группы.

Пользователей можно объединять в группы по разным признакам, например, по должностям, в соответствии с выполняемыми обязанностями или местом в структуре предприятия. Это ускоряет настройку прав доступа - можно настроить их одновременно нескольким пользователям, которые выполняют сходные функции.

Ввод рабочей группы

  • Введите Наименование группы.
  • Введите Участников группы:
  • При редактировании состава группы нужно учитывать, что они входят в состав групп-родителей.
  • Для добавления большого числа пользователей можно использовать команду Подобрать. В этом случае можно одновременно выделить несколько пользователей в списке  и включить их в группу.
  • При использовании команды Еще - Добавить можно включить в группу только одного пользователя.
  • Можно подчинить новую группу одной из уже существующих. Для этого в поле Входит в группу выберите нужную группу из списка.
  • На вкладке Комментарий можно написать подробное описание данной группы.

Права доступа рабочей группы

  • Пользователи, принадлежащие помеченной на удаление рабочей группе, не получают права, назначенные для нее.
  • Пользователи, принадлежащие недействительной рабочей группе, сохраняют свои права.
