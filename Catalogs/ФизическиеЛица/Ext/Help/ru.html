Справочник предназначен для хранения информации обо всех физических лицах.

В справочнике можно объединять физических лиц в произвольные группы или подгруппы.

Для открытия справочника в разделе "Нормативно-справочная информация" в панели навигации в группе "См. также" выполните команду "Физические лица". В списке с помощью команд "Создать" и "Изменить" создавайте и редактируйте карточку физического лица.

Карточки можно указать в справочнике "Пользователи" для расширения сведений о пользователе.

Для каждого физического лица есть возможность указать его личные данные, такие как:

  • пол;

  • дата рождения;

  • информация о документе, удостоверяющем личность (в панели навигации карточки выполните команду "Документы");

  • персональные данные (в панели навигации карточки выполните команду "Персональные данные").

С помощью команды "Все действия - Переместить в группу" можно реорганизовать структуру справочника. Перенос карточек (в списке можно выделить несколько) также выполняется путем перетаскивания мышью из одной группы в другую.
