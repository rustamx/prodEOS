Дескриптор доступа хранит значения правообразующих измерений регистров. Эту информацию можно использовать для детального анализа прав к любому регистру сведений.

Дескрипторы создаются не для каждой записи регистра сведений, а для нескольких записей с одинаковой комбинацией ключевых настроек.

Из карточки дескриптора регистров можно получить следующую информацию:

  • все ключевые настройки, по которым рассчитаны права доступа к регистру (закладка Дескриптор);
  • права, рассчитанные для выбранного регистра (закладка Права доступа).

 

Чтобы отличать похожие дескрипторы друг от друга, используйте данные поля Уникальный идентификатор. Также этот идентификатор поможет определить смену дескриптора доступа того или иного регистра.  

Чтобы пересчитать права по данному дескриптору, нажмите Обновить. Перерасчет выполнится немедленно, даже если права на регистр стояли в очереди отложенного обновления прав.
