Профиль настроек ЭДО

Справочник используется для настройки параметров обмена электронными документами между организацией и контрагентами.

Обмен электронными документами может осуществляться через сервис 1С-ЭДО (несколько операторов), сервис 1С-Такском (оператор Такском) или Напрямую между контрагентами.

Форма представляет собой карточку профиля настроек организации, где отражены основные параметры обмена:

  • Название организации;
  • Идентификатор организации, который был присвоен при подключении обмена;
  • Способ обмена.

На закладке Виды электронных документов указаны виды документов, участвующих в обмене, и версии формата, по которому они будут передаваться.
Флаги Использовать УПД и Использовать УКД помогут автоматически настроить обмен электронными документами.
Гиперссылка Регламент ЭДО позволяет настроить регламент документооборота электронного документа.

При обмене подписывать электронные документы можно только сертификатами, указанными на закладке Сертификаты организации.
Для авторизации на сервере оператора ЭДО может использоваться любой зарегистрированный на сервере сертификат из этого списка.

Для того чтобы проверить корректность выполненных настроек профиля ЭДО, необходимо воспользоваться командой Тест профиля настроек.
