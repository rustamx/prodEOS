Как посмотреть ответы на сообщение

Вы можете сформировать список сообщений таким образом, чтобы просматривать только ответы на какое-то конкретное сообщение. Для этого:

  1. Выберите сообщение из списка.
  2. Выполните команду Показать ответы на сообщение в подменю Все действия, будет сформирован список ответов на выбранное сообщение.

Чтобы вернуться к просмотру всего списка сообщений, еще раз выполните команду Показать ответы на сообщение в подменю Все действия.

Вы также можете настроить, чтобы по умолчанию список сообщений открывался в виде дерева. Для этого выполните команду Изменить вид отображения сообщений в подменю Все действия.

Как начать процесс по сообщению обсуждения

Для организации выполнения работ по сообщению обсуждения Вы можете воспользоваться процессами.

Чтобы стартовать процесс по сообщению обсуждения:

  1. Выполните команду Процесс в подменю Создать на основании, откроется помощник создания процессов.
  2. Выберите нужный процесс из списка.
  3. Заполните карточку процесса.
  4. Нажмите на кнопку Стартовать и закрыть, чтобы начать процесс.

Чтобы посмотреть все процессы и задачи по сообщению:

  1. Откройте карточку нужного сообщения из списка.
  2. Выполните команду Открыть процессы и задачи... , в новом окне откроется список всех процессов и задач по сообщению.

Как начать голосование по сообщению

Для сбора и анализа мнений коллег по вопросу, который вы поднимаете в сообщении обсуждения, начните по нему голосование:

  1. Откройте карточку сообщения обсуждения.
  2. Установите флажок Голосование, станет доступна закладка Настройка голосования.
  3. Введите вопрос, по которому будет проводиться голосование (поле Вопрос голосования).
  4. При необходимости отредактируйте текст вариантов голосования.
  5. Нажмите на кнопку Записать и закрыть, чтобы начать голосование.

После начала голосования, напротив сообщения будет автоматически установлен значок . Посмотреть количество проголосовавших и процентное соотношение между ответами Вы можете в карточке соответствующего сообщения.

Как запретить добавление сообщений по теме

Если Вы хотите запретить дальнейшее добавление сообщений к теме обсуждения, то просто закройте тему (закрыть тему может только ее автор). Для этого:

  1. Перейдите по гиперссылке Тема, в новом окне откроется карточка темы обсуждения.
  2. Установите флажок Закрытая.
  3. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.
