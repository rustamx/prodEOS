В шаблоне внутреннего документа вы можете заполнить реквизиты, и даже список файлов, которые затем будут скопированы в создаваемый по нему документ. Ограничить доступ к шаблону и создаваемому по нему документу вы можете при помощи рабочей группы. Также вы можете заполнить список согласующих лиц для документов.

Как ограничить доступ к шаблону

Ограничить доступ к шаблону вы можете при помощи рабочей группы. Рабочая группа шаблона - это список пользователей, групп пользователей и ролей, у которых есть доступ к шаблону.

Чтобы заполнить рабочую группу, выполните следующие действия:

  1. Перейдите на закладку Рабочая группа шаблона.
  2. Нажмите на кнопку Добавить, станет активным поле выбора. 
  3. Добавьте нового участника из списка или введите с клавиатуры вручную.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.

Если рабочая группа не заполнена, то он будет доступен для просмотра всем пользователям. Редактировать шаблон может только пользователь, который указан в поле Автор.

Как ограничить доступ к документу

Вы также можете ограничить доступ к документу, который будет создаваться по этому шаблону при помощи рабочей группы. Рабочая группа документа - это список пользователей, групп пользователей и ролей, у которых есть доступ к документам, которые создаются по этому шаблону.

Чтобы заполнить рабочую группу, выполните следующие действия:

  1. Перейдите на закладку Рабочая группа документа.
  2. Нажмите на кнопку Добавить, станет активным поле выбора. 
  3. Добавьте участников из списка или введите с клавиатуры вручную.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить изменения.

Чтобы еще быстрее заполнить рабочую группу документа, используйте кнопку Подобрать. В карточке подбора участников рабочей группы можно выбрать не только пользователей и роли, но и воспользоваться автоподстановкой.

Как заполнить список согласующих лиц

Вы  также можете заполнить список согласующих лиц для документов, которые будут создаватся по этому шаблону. Таким образом, при создании документа в его карточку будут добавлены визы без результата на закладку Визы.

Если вы создадите процесс Согласование на основании такого документа, список согласующих лиц будет автоматически скопирован в карточку процесса.

Чтобы заполнить список согласующих лиц, выполните следующие действия:

  1. Перейдите на закладку Согласующие.
  2. Нажмите на кнопку Добавить, станет активным поле выбора.
  3. Выберите согласующее лицо из списка или введите с клавиатуры вручную.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.
