Как создать файл

Для создания нового файла:

  1. Откройте список файлов;
  2. Выберите нужную папку; 
  3. Нажмите клавишу Insert или команду  Создать;
  4. Выберите вариант создания - Из шаблона, Из файла на диске или Со сканера.

Вариант создания файла Со сканера становится доступным, если на компьютере подключена компонента для работы со сканером.

Как контролировать файлы

Файл можно поставить на контроль. При этом будет создана контрольная карточка с записями о том, что контролировать, кого контролировать и на какой срок.

Есть несколько способов работы с контролируемыми файлами:

  • Следить за всеми контрольными карточками в окне На контроле;
  • Контролировать сроки с помощью записей календаря;
  • Получать на электронную почту автоматические уведомления программы об истекающих и приближающихся контрольных сроках.

Контрольную карточку можно снять с контроля , вернуть на контроль , изменить срок контроля , пометить на удаление.

Список ваших контрольных карточек не виден остальным сотрудникам (просмотр разрешен только вам, вашим руководителям и делегатам). Это позволяет использовать любые удобные для вас комментарии к контролируемым файлам.

    Как создать объект на основании одного или нескольких файлов

    На основании одного или нескольких файлов можно создавать документы, процессы и мероприятия. Для этого:

    1. Выделите нужные файлы левой кнопкой мыши + Shift;
    2. Нажмите Создать на основании из панели действий.
    3. К новому документу/мероприятию/процессу выбранные вами файлы прикрепятся автоматически.

    Обратите внимание на принцип присоединения файлов:

    • В процесс файлы прикрепляются ссылками. (Изменение исходного файла отразится и в процессе.)
    • В документ файлы копируются. (Изменения исходных файлов не повлияют на их копии в документе.)

    Как работать с файлами

    Для работы с реквизитами файла (автор, название, описание и т.д.) используется карточка файла, а для работы с самим файлом - соответствующее приложение, установленное на компьютере.

    При редактировании файла его предыдущая версия автоматически сохраняется в информационной базе.

    Файл можно отправить по электронной почте. Для этого в списке файлов или карточке файла следует выполнить команду Отправить по электронной почте. На экран выводится окно для ввода данных письма (получатель, тема, описание, вложенные файлы).

    Если включено использование учета доступа к персональным данным, то в карточке файла появляется список физических лиц, персональные данные которых хранит этот файл. 

    Если файл зашифрован, полнотекстовый поиск по его содержимому не работает.
