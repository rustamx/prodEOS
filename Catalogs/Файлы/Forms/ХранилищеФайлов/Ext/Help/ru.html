Для просмотра списка файлов в разделе "Документы и файлы" в панели навигации выполните команду "Файлы". Команда показывается, если настройка "Использовать хранилище файлов" установлена.

Все файлы хранятся в структуре папок, которую удобно вести для документооборота предприятия. Для отключения просмотра списка файлов в виде структуры следует нажать кнопку "Использовать иерархию". При этом кнопка становится "отжатой", таблица папок не показывается, а в списке файлов показываются файлы всех папок. Для просмотра файлов в структуре следует нажать кнопку "Использовать иерархию". При этом кнопка становится "нажатой", показывается таблица папок, а в списке файлов показываются только файлы текущей папки.

Зеленым цветом отображаются файлы, занятые текущим пользователем для редактирования, а серым цветом - файлом, занятые другими пользователями программы.

Права доступа к папкам файлов настраиваются администратором или пользователем, имеющим право на управление правами той или иной папки.

Для загрузки файлов с жесткого диска используйте загрузку файлов или загрузку папки.

Файл можно открыть для просмотра (выполните команду "Все действия - Просмотреть"). При этом, вносить изменения в файл нельзя.

Для редактирования файла следует выполнить команду "Все действия - Редактировать". Файл открывается и доступен для изменения.

Для открытия карточки файла следует выполнить команду "Все действия - Изменить".

Для переноса файла в другую группу следует выполнить команду "Все действия - Перенести в папку" или перетащить файл или несколько файлов(допускается множественное выделение) указателем мыши.

На основании одного или нескольких файлов можно создавать документы, процессы и мероприятия. Подробнее здесь.
