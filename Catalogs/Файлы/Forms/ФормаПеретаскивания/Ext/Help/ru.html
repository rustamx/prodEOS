Для загрузки файлов с жесткого или сетевого диска в форме хранилища файлов в структуре папок достаточно выделить в Проводнике нужные файлы (и / или папки) и перетащить их с помощью мыши.

Выбранные файлы и папки показываются в списке. Для добавления других файлов повторите перетаскивание.

Если настройка "Удалять файлы после добавления" установлена, загружаемые файлы после добавления в папку будут удалены.

В поле "Описание для создаваемых файлов" можно внести текст описания, который будет добавлен в карточке файлов в поле "Описание".

Если настройка "Хранить версии" установлена, то для каждого файла в карточке файлов будет установлена аналогичная настройка.

Если загружается графический файл (картинка), содержащий текстовую информацию, то при загрузке файла можно выполнить настройку распознавания.

Если загружается текстовый файл, то можно указать кодировку.

В списке выбранных файлов установите флажки для тех файлов, которые требуется загрузить.

Для загрузки отмеченных файлов нажмите кнопку "Загрузить".

Файлы будут загружены с текущую папку хранилища файлов.

Загрузку файлов можно также выполнить с помощью перетаскивания мышью или с помощью команды "Загрузить папку" контекстного меню папки.
