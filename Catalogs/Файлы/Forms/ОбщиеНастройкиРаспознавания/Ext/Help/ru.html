В форме выполняется настройка распознавания. Форма открывается из формы настроек программы.

Настройка Описание
Язык распознавания по умолчанию Указывается язык распознавания. По умолчанию "Русско-английский".
Менять автора нового файла или версии Если не установлена, то автором созданных после распознавания файлов и версий будет тот же пользователь, что и автор оригинального файла изображения.
Пользователь Конкретный пользователь. Если задан, то автором созданных после распознавания файлы и версии будет этот пользователь. Настройка доступна, если Менять автора нового файла или версии  установлена.
Использовать ImageMagick для преобразования отсканированных документов формата PDF Если установлена, то преобразует в изображения и распознает отсканированные изображения формата pdf с помощью программы ImageMagick (на сервере должны быть установлены программы ImageMagick и Ghostscript).
Путь к программе преобразования из PDF Путь к исполняемому файлу программы ImageMagick (по умолчанию "convert.exe"). Если это имя конфликтует с другими программами, нужно указать полный путь, вида C:\Program Files\ImageMagick-6.6.3-Q8\convert.exe.
