Как работать с файлами

Для работы с файлами используйте команды, расположенные на командной панели карточке файла: 

Команда Описание
 Редактировать (F4) Помечает файл как занятый для редактирования и открывает его внешним приложением.
 Закончить редактирование Обновляет файл в хранилище и снимает с него пометку занятости для редактирования 
 Сохранить версию Сохраняет версию редактируемого файла в хранилище, но оставляет его занятым текущим пользователем для редактирования
 Занять Помечает файл в хранилище как занятый текущим пользователем для редактирования, но не открывает внешнее приложение. Приступить непосредственно к редактированию можно будет позже.
 Отменить редактирование Снимает с файла пометку занятости для редактирования
 Открыть Открывает файл внешним приложением для просмотра или редактирования, в зависимости от того, помечен ли файл как занятый текущим пользователем для редактирования.
 Сохранить как Сохраняет файл в указанный каталог на жестком или сетевом диске 

 

При редактировании файла его предыдущая версия автоматически сохраняется в информационной базе.

Если включено использование учета доступа к персональным данным, то в карточке файла появляется список физических лиц, персональные данные которых хранит этот файл. Если владельцем файла не является "Папка файла", а, например, файл присоединен к входящему документу, то этот список недоступен для изменения вручную, а заполняется автоматически из аналогичного списка владельца. Также при этом будет осуществляться протоколирование доступа к полю Описание файла в журнал регистрации. Не рекомендуется использовать другие поля файла (например, Наименование) для хранения персональных данных физических лиц.

Обратите внимание
Если файл зашифрован, полнотекстовый поиск по его содержимому не работает.

Как создать объект на основании файла

На основании файла можно создавать документы, процессы, письма, записи календаря и мероприятия. Для выполните команду Создать на основании на панели действий.

К новому документу/мероприятию/процессу выбранные вами файлы прикрепятся автоматически.

Обратите внимание на принцип присоединения файлов:

  • В процесс файл прикрепляется ссылкой. (Изменение исходного файла отразится и в процессе.)
  • В документ файл копируется. (Изменения исходных файлов не повлияют на их копии в документе.)

Как поставить файл на контроль

Чтобы поставить файл на контроль, нажмите в командной панели карточки. При этом откроется Контрольная карточка файла, в которой нужно установить срок, указать контролируемое действие и исполнителя.

Если файл уже контролировался ранее, кнопка откроет список всех контрольных карточек когда-либо созданных по файлу. В списке можно посмотреть, кто сейчас контролирует файл, открыть контрольные карточки и, при наличии прав, изменить параметры контроля.

После постановки на контроль рекомендуется:

  • Настроить уведомления об истекающих или пропущенных сроках (в настройках программы),
  • Следить за всеми контролируемыми объектами в окне На контроле.
