Форма открывается при выполнении команды "Распознать", а также из форм загрузки файлов или загрузки папки, сканирования.

Настройка Описание
Стратегия распознавания

Указывается стратегия распознавания. Значение по умолчанию "Поместить только в текстовый образ".

  • Создать новую версию в формате TXT - в том же файле создает и делает активной новую версию с распознанным текстом в формате TXT;
  • Создать новую версию в формате HTML - в том же файле создает и делает активной новую версию с распознанным текстом в формате HTML;
  • Создать новый файл в формате TXT - создает новый файл с распознанным текстом в формате TXT в той же папке с тем же именем;
  • Создать новый файл в формате HTML - создает новый файл с распознанным текстом в формате HTML в той же папке с тем же именем;
  • Поместить только в текстовый образ - не создает новых версий или файлов, в самом файле картинки помещает распознанный текст в текстовый образ. Посмотреть и отредактировать текстовый образ можно с помощью команды "Открыть текстовый образ" в карточке файла.
Язык распознавания Указывается язык распознавания. По умолчанию "Русско-английский".
