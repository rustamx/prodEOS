Как открыть файл в стандартной программе

Вы можете открыть файл в стандартной программе непосредственно из встроенного редактора текстовых файлов. Для этого выполните команду Открыть стандартным приложением подменю Все действия.

Если Вы хотите, чтобы файлы с определенным расширением по умолчанию всегда открывались в стандартном приложении, а не во встроенном редакторе текстовых файлов, выполните следующие действия:

  1. Перейдите в раздел Настройка и администрирование.
  2. Выполните команду Персональные настройки группы команд Сервис панели действий.
  3. В персональных настройках программы перейдите на закладку Работа с файлами.
  4. Впишите расширение текстовых файлов в колонку Расширение таблицы.
  5. В колонке Способ открытия выберите вариант Стандартным приложением. 
  6. Нажмите на кнопку ОК, чтобы сохранить настройку.

Таким образом, Вы можете настроить, чтобы файлы с одними расширениями открывались во встроенном редакторе текстовых файлов, а с другими в стандартном приложении.

Как изменить кодировку текста

Если после открытия текстового файла Вы увидели набор бессмысленных знаков, то, вероятнее всего, дело в кодировке и ее надо поменять. Основными кодировками для работы с текстовыми файлами на русском языке являются:

  • IBM866 (Кириллица DOS),
  • KOI8-R (Кириллица KOI8-R),
  • Windows-1251 (Кириллица Windows).

Чтобы изменить кодировку текстового файла:

  1. Выполните команду Изменить кодировку подменю Все действия.
  2. Если нужная кодировка отсутствует, снимите флажок Показывать только основные кодировки.
  3. Выберите нужную кодировку из списка.
  4. Нажмите ОК, чтобы изменить кодировку.
