Загрузка папки

Для загрузки папки с жесткого или сетевого диска в форме хранилища файлов в структуре папок следует выбрать нужную папку выполнить команду "Загрузить папку" контекстного меню.

В поле "Папка" указывается папка, в которую будут помещены файлы. При открытии в поле будет указана текущая папка. Если требуется разместить файлы в другой папке, следует изменить значение поля.

Если настройка "Удалять файлы после добавления" установлена, загружаемые файлы после добавления в папку будут удалены.

В поле "Описание для создаваемых файлов" можно внести текст описания, который будет добавлен в карточке файлов в поле "Описание".

Если настройка "Хранить версии" установлена, то для каждого файла в карточке файлов будет установлена аналогичная настройка.

Если загружается графический файл (картинка), содержащий текстовую информацию, то при загрузке файла можно выполнить настройку распознавания.

Если загружается текстовый файл, то можно указать кодировку.

Для загрузки выбранных файлов следует нажать кнопку "Загрузить".

Загрузку файлов можно также выполнить с помощью перетаскивания мышью или с помощью команды "Загрузить файлы" контекстного меню папки.
