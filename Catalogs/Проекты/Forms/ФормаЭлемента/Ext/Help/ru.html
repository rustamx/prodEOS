В карточке проекта вы можете выполнить все работы по организации и выполнению проекта: определить плановую дату начала и окончания, указать трудозатраты на проект в целом, сформировать проектную команду, настроить доступ, составить план проекта, указать индекс нумерации и многое другое.

Как сформировать проектную команду

Проектная команда - это список пользователей и ролей, которые выполняют работы по проекту.

Чтобы сформировать проектную команду, выполните следующие действия:

  1. Перейдите на закладку Проектная команда.
  2. Нажмите на кнопку Добавить, станет активным поле выбора.
  3. Выберите участника из списка или введите с клавиатуры вручную.
  4. При необходимости впишите его роль в проекте в колонке Роль в проекте.

Чтобы еще быстрее сформировать проектную команду, используйте команду Подобрать.

Формирование проектной команды важно для дальнейшей работы с проектными задачами. В качестве исполнителей проектных задач вы можете выбирать только участников проектной команды.

Как настроить доступ к проекту

Вы можете настроить доступ к проекту посредством заполнения рабочей группы. Рабочая группа - это список пользователей, групп пользователей и ролей, которые могут просматривать проект. Если рабочая группа проекта не заполнена, то его могут просматривать все пользователи, для которых не установлены другие ограничения доступа при работе с проектами (например, ограничение по грифу доступа или организации). Только пользователь, который указан в поле Руководитель проекта может изменять проект. 

Чтобы добавить нового пользователя в рабочую группу, выполните следующие действия:

  1. Перейдите на закладку Рабочая группа.
  2. Нажмите на кнопку Добавить, станет активным поле выбора.
  3. Выберите пользователя из списка или введите с клавиатуры вручную.

Чтобы еще быстрее заполнить рабочую группу, используйте команду Подобрать.

Если в карточке вида проекта установлен флажок Автоматически вести состав участников рабочей группы, то участники проектной команды по мере добавления будут автоматически копироваться в рабочую группу. Чтобы отменить эту настройку, установите флажок Запретить автоматическое добавление участников рабочей группы на закладке Рабочая группа.

Как загрузить данные о проекте из Microsoft Project

Вы можете загружать данные о проекте из Microsoft Project и организовать его исполнение в программе. Источником данных для загрузки является файл проекта Microsoft Project, выгруженный в XML. Во время загрузки не переносятся ресурсы (исполнители) задач, календари проекта и проектных задач. Эти данные вы должны настроить самостоятельно (синхронизация происходит по наименованию).

Чтобы загрузить проект из Microsoft Project:

  1. Выполните команду Загрузить из Microsoft Project в подменю Все действия, откроется карточка загрузки проекта.
  2. Заполните все необходимые для загрузки поля, следуйте подсказкам.
  3. Нажмите на кнопку Загрузить, чтобы начать загрузку проекта из Microsoft Project.

Как указать индекс нумерации

Вы можете указать индекс нумерации проекта, который в дальнейшем можно использовать при формировании нумераторов документов. Для этого выполните команду  Индекс нумерации в карточке проекта.

Как поставить проект на контроль

Проект можно поставить на контроль:

  • следить за сроком и ходом выполнения проекта,
  • переносить контрольный срок,
  • контролировать отдельных пользователей,
  • снимать проект с контроля,
  • выводить аналитические отчеты,
  • смотреть, какие проекты находятся на контроле у ваших коллег.

Программа сама отследит истекающие и приближающиеся контрольные сроки и сообщит вам об этом по почте.

Чтобы поставить проект на контроль, нажмите в командной панели карточки проекта. При этом откроется Контрольная карточка проекта, в которой нужно установить срок, указать контролируемое действие и исполнителя.

Если проект уже контролировался ранее, кнопка откроет список всех контрольных карточек когда-либо созданных по проекту. В списке можно посмотреть, кто сейчас контролирует проект, открыть контрольные карточки и, при наличии прав, изменить параметры контроля.

    После постановки на контроль рекомендуется:
  • Настроить уведомления об истекающих или пропущенных сроках (в настройках программы),
  • Следить за всеми контролируемыми документами в окне На контроле.
