Папки внутренних документов служат для упорядочивания данных и ограничения прав доступа к ним.

Для создания структуры папок следует открыть список внутренних документов и установить использование иерархии (команда "Все действия - Использовать иерархию" должна быть нажата). В левой части формы списка внутренних документов будет расположено дерево папок.

Для создания папки используйте команду "Создать", для редактирования - "Исправить".

В поле "Наименование" задается наименование папки в виде произвольной сроки.

В поле "В папке" выбирается папка, в которой будет размещена создаваемая папка.

В поле "Описание" вводится текст описания использования папки (произвольный текст).

За папкой может быть назначен ответственный (по умолчанию заполняется данными текущего пользователя).

После записи данных о папке становится доступно задание прав доступа в папке (распространяется на все документы, размещенные в папке). Для этого следует выполнить команды "Права доступа". На экран выводится форма для настройки доступа.

Права доступа настраиваются администратором программы.

В группе "Папка содержит" в карточке папки отображается информация о количестве входящих в нее документов, а также файлов и версий файлов (в составе документов). По умолчанию отображается информация только о непосредственном содержимом папки (без учета подпапок). Чтобы получить полную информацию, надо нажать кнопку "Подсчитать с учетом подпапок".
