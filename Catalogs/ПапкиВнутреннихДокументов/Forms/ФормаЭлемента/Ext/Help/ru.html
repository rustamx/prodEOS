Как подсчитать содержимое папки с учетом подпапок

Пункт меню Папка содержит (без учета подпапок) дает информацию об объеме содержимого текущей папки. Для просмотра объема содержимого с учетом подпапок:

Выполните команду Подсчитать с учетом подпапок

Как изменить права доступа

Чтобы изменить права доступа, нажмите на кнопку Права, в новом окне можно настроить права доступа к папке. Например, чтобы добавить нового пользователя:

  1. Нажмите на кнопку Права доступа
  2. В открывшемся окне выберите кнопку Добавить
  3. Выберите новое лицо из списка или введите с клавиатуры вручную
  4. Отметьте действия, которые будут доступны новому лицу
  5. Нажмите кнопку Записать и закрыть для завершения

Команда Права доступна только тем пользователям, для которых настроены права доступа на папку.
