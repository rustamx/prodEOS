В этой форме показаны версии, которые подходят для данного правила (например "Расширение=doc"). Надо учитывать, что в процессе работы правила применяются последовательно, поэтому результат (размещение на томе в конкретной группе томов) может отличаться от списка в данной форме.
Например, список правил и результат их отработки может быть такой:

Размер Больше "3 000 000"

 Файлы с размером более 3 Мб, в том числе и с расширением doc

Расширение=doc

 Файлы с расширением doc (размером менее 3 Мб)

По умолчанию

 Все прочие файлы

Таким образом, в процессе работы в группу томов 2 (связанную с правилом "Расширение=doc"), попадут файлы с расширением doc (размером менее 3 Мб). А в форме проверки в списке будут показаны файлы с расширением doc (любого размера).
