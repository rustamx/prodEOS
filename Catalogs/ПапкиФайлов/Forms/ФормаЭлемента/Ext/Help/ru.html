Папки используются для группировки файлов и ограничения прав доступа к ним.

В карточке папки показаны реквизиты папки и сводная информация по размеру ее содержимого. Реквизиты папки можно менять прямо в карточке при наличии прав на изменение.

В поле Рабочий каталог можно указать каталог на локальном диске, в который будут выгружаться файлы и подпапки из папки при просмотре или редактировании.

Сводная информация по количеству и размеру файлов изначально показана только для самой папки без учета подпапок. Чтобы посмотреть информацию с учетом подпапок нажмите Подсчитать с учетом подпапок.

Для перехода к просмотру содержимого папки нажмите Открыть список папок и файлов.

Возможность настроить права доступа к папке включается, если администратор дал пользователю соответствующие права. Для изменения прав доступа нажмите Настроить права.
