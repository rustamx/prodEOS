В карточке проектной задачи содержится полное описание работ по пункту плана проекта. При создании проектной задачи обязательно заполните:

  • наименование,
  • описание,
  • плановую дату начала, окончания и длительности задачи,
  • список исполнителей проектной задачи,
  • плановые трудозатраты исполнителей,
  • список предшественников задачи,
  • график работы (если график не указан, то будет использован график проекта).

Задачу можно сделать вехой. Веха - это точка, которая отмечает важное событие в проекте и используется для контроля за ходом выполнения проекта. Веха имеет нулевую длительность и у нее не может быть исполнителей.

После сохранения в задаче автоматически проставляется код СДР. Код СДР (код структурной декомпозиции работ) - это алфавитно-цифровой код, который определяет место задачи в структуре проекта.

Плановые даты проектных задач рассчитываются автоматически, исходя из длительности задачи, указанных предшественников и ограничений. Также можно указать задержку по отношению к предшественнику и тип ограничения в одноименном поле.

Для организации выполнения проектных задач используйте процессы.
