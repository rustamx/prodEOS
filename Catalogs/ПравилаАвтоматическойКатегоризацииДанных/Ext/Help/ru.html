Правила автоматической категоризации предназначены для автоматического присвоения категорий документам и файлам на основании заранее определенных критериев.

Создание правила

Необходимо заполнить обязательные поля:

  • Категория - выбрать категорию из списка категорий
  • Наименование - наименование правила 
  • Используется - признак того, что правило работает
  • Типы объектов - указать для объектов каких типов правило будет применяться.
  • Условия - задать условия, по которым правило будет применяться к тому или иному документу или файлу.

    Условия в правиле могут быть одного из двух типов: подстроки и вычисляемое выражение.

    Тип правила

    Описание

    Ключевые слова

    В условии задается список строк, разделенных клавишей Enter.

    Условие считается применимым к файлу, если одна из подстрок условия находится внутри содержимого файла или среди значений реквизитов файлов.

    Условие считается применимым к документу, если одна из подстрок условия находится внутри содержимого одного из файлов документа или среди значений реквизитов документов.

    Выражение не встроенном языке

    В условии задается выражение на внутреннем языке 1С. Выражение обязательно должно присваивать значение булевой переменной Результат. Выражение может оперировать переменной Источник, которая содержит ссылку на проверяемый документ или файл.

    Условие считается применимым к документы или файлу, если в результате выполнения выражения переменной Результат было присвоено значение Истина.

    Пример: Условие, вычисляющее файлы MS Excel
    Результат = Ложь;
    Если ТипЗнч(Источник) = Тип("СправочникСсылка.Файлы")
        И ТипЗнч(Источник.ВладелецФайла) = Тип("СправочникСсылка.ПапкиФайлов") Тогда
        Если НРег(Источник.ТекущаяВерсияРасширение) = "xls"     
            ИЛИ НРег(Источник.ТекущаяВерсияРасширение) = "xlsx" Тогда
            Результат = Истина;
        КонецЕсли;
    КонецЕсли;

    Примечание: выражение, записанное в условии с типом "Вычисляемое выражение", выполняется в Безопасном режиме.

    В правиле категоризации могут одновременно присутствовать несколько условий разного типа. Правило считается применимым к документу или файлу, если применимо одно из его условий.

    Список правил

    Список правил автоматической категоризации открывается командой "Правила категоризации" в разделе "НСИ". Для каждого правила доступна команда "Проверить правило", которая открывает форму проверки правила.

    Проверка правил

    Правило считается применимым к документу или файлу, если применимо одно из его условий.
