Правила автозаполнения шаблонов файлов содержат настройки подстановок данных, которые выполняются при сохранении созданных из шаблонов файлов.

Для просмотра списка настроек автозаполнения в разделе "Нормативно-справочная информация" необходимо выполнить команду "Правила автозаполнения шаблонов файлов". На экран выводится список настроек автозамены.

Для создания документа используйте команду "Создать", для редактирования - "Изменить".

Для настройки автозаполнения следует задать:

  • Наименование правила
  • Шаблон файла, к которому применяется правило
  • Тип документа, к которому применимо правило (Исходящий либо Внутренний документ) 
  • Владелец файла, к которому применимо правило. Список доступных к выбору значений определяется значением поля Тип владельца. Правило может применяться к папке файлов, виду внутреннего или исходящего документа. 
  • Список замен в тексте файла-шаблона в следующем формате:

Какое поле заменить

Какую строку заменить

На что заменить

Формат

Выражение

Задает имя поля в файле-шаблоне  Microsoft Word либо имя переменной в файле-шаблоне Open Office Writer. Задает строку, которую необходимо заменить в файле-шаблоне.

Строковая константа либо имя реквизита файла либо документа, к котороуму прикреплется файл. строковая константа задается в формате Значение, имя реквизита - в формате {Значение}.

 Задает формат представления значения реквизита, имя которого указано в колонке "На что заменить". Если в качестве значения, которое подставляется в документ выбран результат выполнения выражения, то в данной колонке отображается текст указанного выражения.

Для текстовых документов (*.txt, *.html, *.htm) нельзя вводить поле "Какое поле заменить".

Основные и дополнительные реквизиты файла и владельца файла удобно указывать, воспользовавшись кнопкой выбора в соответствующем поле на форме настройки замены.

Для пары значений Шаблон файла-Владелец файла возможно задание только одного правила автозаполнения.

Примеры задания настроек автозаполнения:

  • Заполнение поля "Автор документа" значением реквизита владельца файла

    Какое поле заменить

    Какую строку заменить

    На что заменить

     АвторДокументаИмя

    {ВладелецФайла.Подготовил}

    Примечание: если в поле "На что заменить" ввести значение ВладелецФайла.Подготовил без символов { и }, то в поле АвторДокументаИмя файла запишется строка "ВладелецФайла.Подготовил"

  • Замена в файле строки "Автор документа" на значение реквизита владельца файла

    Какое поле заменить

    Какую строку заменить

    На что заменить

    <Автор документа>

    {ВладелецФайла.Подготовил}


  • Замена в файле строки "Дата создания" значением реквизита файла "Дата создания" в формате "ДД.ММ.ГГГГ"

    Какое поле заменить

    Какую строку заменить

    На что заменить

    <<ДатаСоздания>>

    {ДатаСоздания}, ДФ = 'dd.MM.yyyy'


    Пользователь, если он не работает в режиме web-клиента, имеет возможность проведения интерактивной проверки правила автозаполнения файлов. Для этого необходимо нажать на кнопку "Проверить правило" на форме редактирования правила. Система предлагает пользователю выбрать документ, данные которого затем используются для заполнения шаблона. При выполнении проверки правила никакие изменения в информационную базу не заносятся.
