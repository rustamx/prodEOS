Для ознакомления и облегчения работы с ключевыми объектами программы, например, документами, процессами и задачами, можно использовать специальные инструкции.

Инструкции отображаются в карточках объектов и подсказывают, какие действия и в каком порядке необходимо выполнить, чтобы исполнить поручение, поставить задачу и т. д.

Как создать новую или отредактировать существующую инструкцию

Инструкции - это обычные HTML-файлы, которые можно создавать и редактировать любым удобным способом. Права на работу с инструкциями есть только у сотрудников, ответственных за ведение нормативно-справочной информации.

Чтобы создать инструкцию, выберите в списке предмет, к которому должна относиться инструкция и нажмите кнопку  . 

Для редактирования текста инструкции нужно выгрузить его из программы при помощи кнопки Сохранить на диск.

Например, перед использованием типовых инструкций, в названии которых присутствует слово "шаблон", требуется заменить синий текст в инструкции на собственную информацию.

Текст можно подготовить в любом HTML-редакторе (например, в редакторе "1С:Предприятия") и загрузить в конвфигурацию при помощи кнопки Загрузить файл инструкции.

Как указать карточку, на которой будет отображаться инструкция

Чтобы инструкция отображалась на заданной карточке, ее нужно отнести к соответствующему предмету. 

Инструкции, отнесенные к родительскому предмету, будут показаны для всех дочерних объектов.

Например, чтобы инструкция показывалась при работе с внутренними документами, она должна отнесена к предметам Внутренние документы или Документы, а инструкция для журнала передачи - к предмету Журнал передачи.

Чтобы отнести инструкцию к определенному предмету, выберите нужный предмет в соответствующем поле.

Например, если для инструкции "Как обращаться с документами с грифом ДСП" выбрать предмет Документы, то она будет отображаться для внутренних, входящих и исходящих документов.

Как управлять показом инструкции

Можно задавать условия показа инструкции, ограничивать к ней доступ или отключать.

Например, можно отображать инструкцию только для некоторых сотрудников, или определенных категорий документов (договоров, или внутренних документов, находящихся в процессе рассмотрения) и т. д.

Параметры отображения задаются индивидуально для каждой инструкции в ее карточке.

Отключение инструкции

Для быстрого управления показом служит флажок Активна: если он установлен, то инструкция будет отображаться, если сброшен - нет. Отключенные инструкции выделены в списке серым цветом.

Настройка условий отображения

Можно задать любую комбинацию условий, при выполнении которых инструкция будет показана пользователю:

Условие Пример использования
Организация Для разделения инструкций, предназначенных для разных организаций
Вид документа
(только для инструкций к документам)		 Показывать инструкцию только для многосторонних договоров
Гриф доступа
(только для инструкций к документам)		 Показывать только для документов ДСП
Состояние документа
(только для инструкций к документам)		 Только для согласованных
Вопрос деятельности
(только для инструкций к документам)		 Показывать только для документов по договорной деятельности

В примере из таблицы инструкция будет открываться только в карточках согласованных многосторонних договоров, имеющих гриф для служебного пользования.

Ограничение доступа

Инструкции можно показывать конкретным пользователям, подразделениям или исполнителям определенных ролей.

Для составления списка сотрудников, допущенных к просмотру, служит кнопка Добавить в разделе Доступ.

По умолчанию список незаполнен, и инструкция доступна всем пользователям программы.
