В карточке можно просматривать и задавать реквизиты и условия для папки поиска.

Для поиска можно задать следующие условия:

  • Тема – введите часть темы письма,
  • Переписка с – будет найдены письма от указанных адресатов или отправленные указанным адресатам, 
  • От кого – укажите корреспондента(ов), от которого пришло письмо,
  • Кому – укажите корреспондента(ов), которому отправлено письмо,
  • За период – укажите период отправки/получения письма,
  • Важность – укажите важность письма – обычное, важное, не важное,
  • Флаг – укажите цвет флага,
  • Комментарий – укажите часть текста комментария к письму,
  • Проект – если ведется учет по проектам, можно указать проект для письма,
  • Имеет вложения – можно искать письма только с вложениями,
  • Прочтено – можно искать только в  прочтенных письмах,
  • Есть ответ – можно искать только в письмах, на которые есть ответ,
  • Переслано – можно искать только в письмах, которые пересылались,
  • Входящие письма – можно искать только во входящих письмах,
  • Исходящие письма – можно искать только в исходящих письмах,
  • Папка – укажите папки для поиска, по умолчанию поиск ведется только в Моих папках.
  • Учетная запись – если используется несколько учетных записей, можно указать в каких именно искать
  • Удалено – можно использовать поиск в удаленных письмах,
  • Выбрать первые – укажите количество выводимых результатов  поиска, по умолчанию 100,
  • Соединять условия – вариант И – в список выводятся письма, для которых все заданные условия соблюдаются, вариант ИЛИ – хотя  бы одно условие соблюдается.
