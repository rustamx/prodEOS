Загрузка файлов

В данном окне вы можете подобрать один или несколько файлов для загрузки.

Чтобы добавить файл в список, нажмите Выбрать и укажите путь к файлу. Если вы хотите стереть выбранные файлы с диска, установите флажок Удалять файлы после добавления.

Для создаваемых документов следует указать следующие параметры:

  • папка - указывается папка, в которую будет помещен документ;
  • вид документа - указывается вид внутреннего документа, показывается, если настройка программы Использовать виды внутренних документов установлена;
  • организация - указывается организация в случае ведения документооборота для нескольких организаций (показывается, если настройка программы "Использовать учет по организациям" установлена);
  • гриф доступа - если установлена настройка программы "Использовать грифы доступа", то нужно обязательно присвоить гриф доступа, выбрав его из числа заранее настроенных администратором грифов. Гриф определяет права доступа пользователей к внутреннему документу;
  • вопрос деятельности - показывается, если настройка программы "Использовать вопросы деятельности" установлена;
  • утвердил - пользователь, который утвердил документ;
  • подготовил - пользователь, который подготовил документ.

Если установлен флажок "Регистрировать создаваемые документы", то документы будут созданы, они будут зарегистрированы (присвоен номер) и установлено состояние Зарегистрирован.
Если флажок не установлен, но в поле "Утвердил" указан пользователь, который утвердил документ, то документы будут созданы, им будет присвоено состояние Утвержден, если пользователь не указан, то документы будут созданы с состоянием Проект.

Для создания документа (документов, если выбрано несколько файлов) следует нажать кнопку "Загрузить". По каждому загруженному файлу будет создан внутренний документ, наименование которого совпадает с наименованием файла.
