Как работать с карточкой внутреннего документа

В блоке Хранение следует заполнить значение поля состав описывающее количество экземпляров и листов документа. Состав заполняется через специализированную форму. Если документ помещен (списан) в дело, то указывается данные дела. При списании документа в дело подсчет общего числа листов выполняется по значениям полей Количество листов и Листов в приложениях в карточке документа.
Поле учета состава документа показываются, если настройка программы Использовать состав документов установлена.

Если настройка программы Использовать учет доступа к персональным данным включена, то в карточке внутреннего документа на закладке Дополнительно появляется список физических лиц, персональные данные которых хранит этот документ. Этот список физических лиц будет автоматически распространен на все файлы, присоединенные к данному документу и их версии. Также при этом будет осуществляться протоколирование доступа к полю Содержание внутреннего документа в журнал регистрации. Не рекомендуется использовать другие поля внутреннего документа (например, Наименование или Комментарий) для хранения персональных данных физических лиц.

Если документ был только что создан, на карточке документа появляется кнопка  Заполнить по шаблону . При нажатии на нее открывается форма выбора шаблонов документов данного типа, если для данного типа документов в системе есть шаблоны.

Если настройка программы Использовать автозаполнение шаблонов файлов включена, то в списке файлов внутреннего документа становится доступной команда Заполнить файл данными документа, при выполнении которой осуществляется попытка заполнить файл данными документа, основываясь на правилах автоматического заполнения шаблонов файлов.

Как подписать документ ЭП

Для подписания документа выполните команду Документ - Подписать в подменю Еще карточки документа.

Обратите внимание, что при подписании документа автоматически подпишутся и все его файлы. Чтобы подписать только прикрепленный к документу файл, воспользуйтесь командой ЭП и шифрование в контекстном меню списка файлов.

Подписанный документ или файл недоступен для редактирования. И для внесения изменений придется удалить все подписи.

Для использования ЭП на вашем рабочем компьютере должно быть установлено криптографическое ПО и хотя бы один личный сертификат ЭП.

Нужен ли документу с ЭП бумажный оригинал?

Согласно 63-ФЗ "Об электронной подписи" при подписании усиленной ЭП можно использовать оригинал документа как в бумажном, так и в электронно виде. Поэтому порядок работы с электронными документами определяется только соглашением между участниками электронного обмена. В соглашении должен быть четко прописан порядок работы с каждым видом документов: в каких случаях требуется бумажный оригинал, а в каких нет.

Как настроить срок действия документа

Для настройки срока действия документа необходимо в карточке изменения срока действия документа заполнить реквизиты:

  • Действует с - дата начала действия договора,
  • Действуют по - указывается дата окончания действия, если она известна заранее, 
  • Бессрочный - устанавливается, если дата окончания договора неизвестна,
  • Продление - порядок продления договора.

Карточку изменения срока действия документа можно открыть при помощи клавиши F4 в поле Срок действия.

Посмотреть историю изменения срока действия документа Вы можете одним из двух способов:

  • Способ 1 – команда Карточка документов подменю Печать.
  • Способ 2 – команда История в карточке изменения срока действия.

Изменение возможно только для тех документов, в карточке вида которых установлен флажок Учитывать срок действия (при установке флажка Является договором устанавливается автоматически).

Права доступа к внутренним документам определяются папкой, в которой размещен документ, и настраиваются администратором программы при необходимости.

Как контролировать внутренние документы

Внутренний документ можно поставить на контроль. При этом будет создана контрольная карточка с записями о том, что контролировать, кого контролировать и на какой срок.

Есть несколько способов работы с контролируемыми документами:

  • Следить за всеми контрольными карточками в окне На контроле;
  • Контролировать сроки с помощью записей календаря;
  • Получать на электронную почту автоматические уведомления программы об истекающих и приближающихся контрольных сроках.

Список ваших контрольных карточек не виден остальным сотрудникам (просмотр разрешен только вам, вашим руководителям и делегатам). Это позволяет использовать любые удобные для вас комментарии к контролируемым документам.

Как прикрепить один или несколько файлов

К каждому внутреннему документу можно прикрепить один или несколько файлов. Удобнее всего "перетащить" выбранные файлы в список Файлы на закладке Обзор.

При перетаскивании файла в карточку, где уже есть файл с таким именем, программа предложит сохранить версию имеющегося файла либо не даст провести операцию, если имеющийся файл новее перетаскиваемого.

Для присоединения к внутреннему документу второго файла с таким же именем воспользуйтесь одним из следующих способов:

  • переименуйте имеющийся файл и добавьте второй файл любым из возможных способов.
  • присоедините второй файл, используя кнопку Добавить.

Как настроить рабочую группу

Доступ к документу можно ограничить указанным списком лиц на закладке Рабочая группа.

Рабочая группа – это список пользователей и ролей, которые могут просматривать этот документ. Заполнить рабочую группу может любой пользователь, у которого есть права на просмотр и изменение документов. Если рабочая группа не заполнена, то документ могут просматривать все пользователи.

При заполнении обязательно внесите текущего пользователя, иначе программа не даст записать документ.

Если для вида документа указано Автоматически вести состав рабочей группы, то в рабочую группу автоматически добавляются:

  • авторы процессов по документу,
  • исполнители процессов по документу,
  • пользователи, указанные в самом документе.

Для отключения автоматического добавления участников в рабочую группу конкретного документа установите флажок Запретить автоматическое добавление участников рабочей группы на закладке Рабочая группа.
