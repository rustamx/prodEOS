Список Настройки почты содержит перечень учетных записей электронной почты, которые сервер СВД использует для отправки сообщений получателям

Чтобы создать новую настройку, выполните следующие действия:

  1. Нажмите на кнопку Создать командной панели, в новом окне откроется карточка настройки почты.
  2. Введите название настройки в поле Наименование.
  3. Выберите учетную запись электронной почты из списка или введите с клавиатуры вручную.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.
