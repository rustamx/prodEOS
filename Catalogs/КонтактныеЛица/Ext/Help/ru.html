В справочнике "Контактные лица" осуществляется учет информации о контактном лице, используемой для справочника "Контрагенты".

Для открытия списка контактных лиц в справочнике "Контрагенты" в панели навигации выполните команду "Контактные лица". В списке можно ввести данные о сотрудниках, через которых осуществляется контакт с выбранным контрагентом.

Для каждого контактного лица указываются:

  • Фамилия, Имя, Отчество;
  • Карточка физического лица;
  • Должность;
  • Контактная информация (телефоны, адрес электронной почты);
  • Комментарий.
