Виды исходящих документов настраиваются ответственным за НСИ в соответствии с правилами делопроизводства и документооборота, установленными в Вашей организации.

Вид определяет правила формирования регистрационного номера документов, нормативный срок их исполнения и набор  дополнительных реквизитов и свойств.

Для настройки видов в разделе "Нормативно-справочная информация" в панели навигации выполните команду "Виды документов", в открывшейся форме выберите "Исходящие документы".

В списке видов документов создайте перечень нужных видов (например, "Бухгалтерский отчет", "Договор", "Приказ по кадрам" и т.д.).

Справочник видов - иерархический справочник. Это позволяет сгруппировать отдельные виды документов в группы. В поле "Группа" указывается группа, в которую входит вид документа.

В поле "Нормативный срок исполнения" указывается срок в днях, в течении которого поручения по документам данного вида следует исполнять.

На закладке "Настройки" выполняется следующая настройка:

  • Использовать срок ответа - для некоторых видов документов не требуется учитывать срок ответа. С помощью данной настройки учет срока исполнения может быть отключен. Если снят, то в карточке документа данного вида станет невидимым поле "Срок  ответа";
  • Учитывать входящий номер и дату получателя - при установке в карточке документа данного вида появятся реквизиты "Вх. номер, присвоенный получателем", "Вх. дата, присвоенная получателем";
  • Вести учет по номенклатуре дел - если установлен, то в карточке документа появится реквизит "Номенклатура дел".
    В нумераторы включена возможность использовать индекс по номенклатуре дел в регистрационном номере документа. Индекс по номенклатуре дел выбирается из реквизита "Номенклатура дел" карточки документа данного вида, поэтому для использования индекса по номенклатуре дел должна быть включена данная настройка.
  • Необходима печать штрихкода при создании документа - если установлен, то при закрытии карточки только что созданного исходящего документа данного типа будет осуществляться проверка того, был ли напечатан штрихкод созданного документа (любым из возможных вариантов). Если штрихкод не был напечатан, то система выдаст предупреждение с запросом подтвердить закрытие формы.
  • Является комплектом документов - данный признак означает, что документы с этим видом являются комплектами других документов и файлов. Установить данный признак можно при условии что не установлен признак "Вести учет по номенклатуре дел".
  • Автоматически вести состав участников рабочей группы - для документов данного вида в рабочую группу будут автоматически добавляться новые участники рабочих процессов по документу.
  • Для документов этого вида заполнение рабочей группы является обязательным - для документа данного вида контролируется заполнение рабочей группы.


На закладке "Нумерация" настраиваются правила нумерации документов данного вида. В каждой строке можно указать, как нумеровать документы - вручную или автоматически, какой использовать нумератор для автоматической нумерации, а также набор измерений, для которых действует указанная настройка нумерации. Например, для вида документа "Исходящее письмо" можно настроить один нумератор для подразделения "Отдел кадров" и другой для "Бухгалтерии". Каждый из них может иметь свой формат номера и периодичность нумерации.
