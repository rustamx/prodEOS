Что такое дескриптор доступа

Дескриптор доступа предназначен для хранения информации о правах файлов.

Дескрипторы создаются не для каждого файла, а для комбинации ключевых настроек. Например, для сотни файлов с одинаковым владельцем будет использован один дескриптор.

Информацию, полученную из карточки дескриптора, можно использовать для детального анализа прав файлов.

Как работать с карточкой дескриптора

Из карточки дескриптора можно получить следующую информацию:

  • дескриптор владельца, по которому рассчитаны права доступа к выбранному файлу (закладка Дескриптор);
  • данные о том, какие объекты используют данный дескриптор (закладка Объекты);
  • права, рассчитанные для выбранного файла (закладка Права доступа).

Чтобы отличать похожие дескрипторы друг от друга, используйте данные поля Уникальный идентификатор. 

Чтобы пересчитать права по данному дескриптору, нажмите Обновить на закладке Права доступа. Перерасчет выполнится немедленно, даже если права на файл стояли в очереди отложенного обновления прав.
