Для быстрого поиска нужных сообщений, Вы можете предварительно отсортировать их список. Например, выбрать просмотр только прочтенных или не прочтенных сообщений, а также только сообщений по конкретному автору. Также Вы можете воспользоваться полнотекстовым поиском в данных.

Как создать новое сообщение

Вы можете создать сообщение обсуждения только в рамках какой-либо темы, существующей или новой.

Чтобы создать новое сообщение в теме обсуждений, выполните следующие действия:

  1. Установите отбор по одной теме обсуждения. В режиме просмотра списком это можно сделать с помощью быстрого отбора Темы и разделы. В режиме просмотра По темам и разделам это можно сделать, выбрав в списке тем конкретную тему.
  2. Нажмите на кнопку Написать, в новом окне откроется карточка нового сообщения.
  3. Введите текст сообщения, при необходимости прикрепите файл, который поясняет суть сообщения.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить сообщение.

Чтобы создать новое сообщение в ответ на другое сообщение, выполните следующие действия:

  1. Выберите сообщение из списка.
  2. Нажмите на кнопку Ответить, в новом окне откроется карточка нового сообщения.
  3. Введите текст сообщения, при необходимости прикрепите файл, который поясняет суть сообщения.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить сообщение.

Как начать процесс по сообщению обсуждения

Для организации выполнения работ по сообщению обсуждения используйте процессы.

Чтобы создать новый процесс по сообщению обсуждений:

  1. Выберите сообщение из списка.
  2. Выполните команду Процесс подменю Создать на основании, в новом окне откроется помощник создания процессов.
  3. Выберите нужный процесс из списка, заполните все необходимые для старта поля в его карточке.
  4. Нажмите на кнопку Стартовать и закрыть, чтобы начать процесс по сообщению.

Чтобы просмотреть все процессы и задачи по сообщению:

  1. Выберите сообщение из списка, в новом окне откроется карточка сообщения.
  2. Выполните команду Открыть процессы и задачи... подменю Все действия, в новом окне откроется список процессов и задач.
