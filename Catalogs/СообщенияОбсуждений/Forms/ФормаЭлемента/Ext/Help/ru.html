Как начать голосование

Если Вы хотите узнать отношение коллег к Вашему сообщению, начните голосование по нему:

  1. Установите флажок Голосование, станет активной закладка Настройка голосования.
  2. Введите вопрос голосования (поле Вопрос голосования).
  3. При необходимости отредактируйте варианты ответа, которые установлены по умолчанию.
  4. Нажмите на кнопку Записать и закрыть, чтобы начать голосование.

После начала голосования напротив сообщения появится значок  и станет невозможно изменить вопрос или результат голосования без удаления результатов ранее проголосовавших пользователей.

Как изменить результат своего голосования

Для изменения результата своего голосования выполните следующие действия:

  1. Выберите нужный вариант из списка.
  2. Нажмите на кнопку Проголосовать, чтобы изменить результат голосования.

Если Вы не хотите больше участвовать в голосовании, используйте команду Отменить голос.

Как начать процесс по сообщению

Для организации выполнения работ по сообщению обсуждения, воспользуйтесь процессами.

Чтобы стартовать процесс по сообщению обсуждения:

  1. Выполните команду Процесс подменю Создать на основании, откроется помощник создания процессов.
  2. Выберите нужный процесс из списка.
  3. Заполните карточку процесса.
  4. Нажмите на кнопку Стартовать и закрыть, чтобы начать процесс.

Чтобы посмотреть все процессы и задачи по сообщению, выполните команду Открыть процессы и задачи... , в новом окне откроется список всех процессов и задач по сообщению.
