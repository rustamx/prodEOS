Для просмотра списка всех версий файла откройте его карточку (F2 или команда "Открыть карточку") и перейдите в раздел "Версии файлов".

В списке версий можно:

  • открыть файл версии соответствующим приложением для просмотра (команда "Открыть");
  • удалить ненужные версии (клавиша Del или команда "Пометить текущий элемент на удаление");
  • изменить активную (текущую) версию (команда "Сделать активной");
  • сравнить две версии (команда "Показать отличия"). Сравнение версий поддерживается для файлов HTML, TXT, RTF, DOC, DOCX, ODT.

 

Если включено использование учета доступа к персональным данным, то в карточке версии появляется список физических лиц, персональные данные которых хранит файл, которому принадлежит данная версия. Этот список недоступен для изменения вручную, а заполняется автоматически из аналогичного списка у файла.
