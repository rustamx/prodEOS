Чтобы создать новый транспорт отправки сообщений, выполните следующие действия:

  1. Дайте транспорту подходящее название в поле Наименование.
  2. Выберите способ отправки сообщений из списка или введите с клавиатуры вручную, при необходимости создайте новый.
  3. Выберите формат сообщения из списка, для выбора доступны следующие варианты:
    • ГОСТ Р 53898-2010 - сообщение записывается согласно требованиям, ГОСТа;
    • Сообщение 1С:Документооборот - сообщение также записывается по ГОСТ Р 53898-2010, но с возможностью добавления ЭП.
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.
