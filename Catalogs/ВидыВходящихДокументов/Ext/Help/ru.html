Виды входящих документов настраиваются ответственным за НСИ в соответствии с правилами делопроизводства и документооборота, установленными в Вашей организации.

Вид определяет правила формирования регистрационного номера документов, нормативный срок их исполнения и набор  дополнительных реквизитов и свойств.

Для настройки видов в разделе "Нормативно-справочная информация" в панели навигации выполните команду "Виды документов", в открывшейся форме выберите "Входящие документы". В списке видов документов создайте перечень нужных видов (например, "Бухгалтерский отчет", "Договор", "Приказ по кадрам" и т.д.).

На закладке "Настройки" выполняется следующая настройка:

  • Использовать срок исполнения - для некоторых видов документов не требуется учитывать срок исполнения. С помощью данной настройки учет срока исполнения может быть отключен. Если снят, то в карточке документа данного вида станет невидимым поле "Срок  исполнения";
  • Вести учет по номенклатуре дел - если установлен, то в карточке документа данного вида появится реквизит "Номенклатура дел".
    В нумераторы включена возможность использовать индекс по номенклатуре дел в регистрационном номере документа. Индекс по номенклатуре дел выбирается из реквизита "Номенклатура дел" карточки документа данного вида, поэтому для использования индекса по номенклатуре дел должна быть включена данная настройка.
  • Необходима печать штрихкода - если установлен, то при закрытии карточки только что созданного входящего документа данного типа будет осуществляться проверка того, был ли напечатан штрихкод созданного документа (любым из возможных вариантов). Если штрихкод не был напечатан, то система выдаст предупреждение с запросом подтвердить закрытие формы.
  • Является обращением от граждан - данный признак означает, что документы с этим видом являются обращением от граждан и имеют актуальный именно для них состав реквизитов – вид обращения, повторное, дубликат и т.п.
  • Учитывать сумму документа - если флажок установлен, то для документов данного вида можно будет указать сумму и валюту документа. 
  • Автоматически вести состав участников рабочей группы - для документов данного вида в рабочую группу будут автоматически добавляться новые участники рабочих процессов по документу.
  • Для документов этого вида заполнение рабочей группы является обязательным - для документа данного вида контролируется заполнение рабочей группы.
  • Подписывать резолюции электронной подписью - если флажок установлен, то для документов данного вида утверждение каждой резолюции ЭП будет обязательным.


На закладке "Нумерация" настраиваются правила нумерации документов данного вида. В каждой строке можно указать, как нумеровать документы - вручную или автоматически, какой использовать нумератор для автоматической нумерации, а также набор измерений, для которых действует указанная настройка нумерации. Например, для вида документа "Входящее письмо" можно настроить один нумератор для подразделения "Отдел кадров" и другой для "Бухгалтерии". Каждый из них может иметь свой формат номера и периодичность нумерации.
