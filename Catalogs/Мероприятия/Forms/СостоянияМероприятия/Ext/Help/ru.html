Данная форма предназначена для отображения и ручного изменения состояний документа. Состояние определяет текущее положение документа в жизненном цикле, а также влияет на доступность полей и команд в карточке документа. 

Документ может иметь одновременно до пяти состояний - по одному из каждой группы (согласование, утверждение, регистрация, рассмотрение, исполнение).
Состояние можно установить вручную либо автоматически (при старте и выполнении процессов).

Для указания состояния вручную выберите нужное состояние в одном из полей. Для состояний установленных вручную выводится кто и когда это состояние установил. Если состояние установлено процессом, то для перехода к этому процессу воспользуйтесь гиперссылкой рядом с полем состояния.

Для просмотра полной истории изменения состояния перейдите на  закладку "История состояний".
