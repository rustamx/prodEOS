Ознакомление с результатом согласования

По инициированному вами согласованию принято решение. Ознакомьтесь с ним в поле Результат согласования.

  • Если результат Не согласовано, завершите (при этом предмет останется не согласован) или повторите согласование (стартует следующий цикл согласования).
  • Если Согласовано - просто ознакомьтесь с листом согласования и историей выполнения процесса.

Лист согласования с визами всех исполнителей можно распечатать, сохранить на диск или отправить почтой. Лист согласования содержит результаты последнего цикла согласования.

Если согласующие лица оставили для вас развернутый комментарий, с ним удобнее ознакомиться в карточке задачи исполнителя. Там же, при необходимости, можно задать вопрос исполнителю. Чтобы открыть карточку задачи, дважды кликните по исполнителю в списке согласующих лиц.

Как задать вопрос по задаче

  1. Перейдите по ссылке Задать вопрос (если по задаче уже задан хотя бы один вопрос, ссылка будет иметь вид - Вопросы по задаче).
  2. При необходимости, прикрепите дополнительные файлы и установите срок ответа.

Если ссылка недоступна, проверьте, установлена ли настройка программы Использовать рассмотрение вопросов выполнения задач.

Вы можете остановить или прервать отправленный вопрос соответствующими командами в командной панели его карточки.
