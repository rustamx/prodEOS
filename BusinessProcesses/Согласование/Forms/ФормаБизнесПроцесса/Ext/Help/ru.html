Как установить персональные сроки для каждого согласующего

Установите флажок Разные сроки в командной панели списка согласующих. После этого:

  • Если работа по задаче еще не начата, установите сроки в колонках Дней и Часов;
  • Если работа по задаче уже ведется, сроки устанавливаются через кнопку Изменить срок.

Если оба способа изменения сроков недоступны, значит: либо работа по задаче завершена и изменить ее срок невозможно, либо не установлена настройка программы Вести учет переносов сроков задач.

Разница между остановкой и прерыванием процесса

Остановленный процесс можно возобновить (команда Продолжить). Прерванный - нельзя.

При остановке процесса: на "паузу" ставится сам процесс и все его вложенные и подчиненные процессы (с задачами). Остановленные задачи не отображаются в списках задач.

При прерывании: прерывается сам процесс, все его подчиненные и вложенные процессы (с задачами), состояние предмета "Согласован" очищается. Действие необратимо.

Как управлять запуском процесса

Для процесса можно настроить:

  • Отложенный старт - запуск процесса через определенный промежуток времени. Например, сразу после заключения договора создать процесс "Согласовать отчет об отгрузке" и отложить его запуск на месяц. Инструкция по настройке отложенного старта.  
  • Повторение - запускает процесс по указанному вами расписанию. Например, согласование оплаты за канцтовары в последнюю пятницу каждого месяца. Инструкция по настройке повторения.

Как контролировать процесс

Процесс можно поставить на контроль  (если включена настройка программы Использовать контроль объектов). При этом будет создана контрольная карточка с записями о том, что контролировать, кого контролировать и на какой срок.

Есть несколько способов работы с контролируемыми процессами:

  • Следить за всеми контрольными карточками в окне На контроле;
  • Контролировать сроки с помощью записей календаря;
  • Получать на электронную почту уведомления об истекающих и приближающихся контрольных сроках.

Как сделать подписание виз ЭП обязательным

Визу можно подписать электронной подписью. При этом виза становится недоступна для редактирования. Изменить визу можно будет только после удаления всех ее ЭП.

В зависимости от правил документооборота на предприятии, подписание виз ЭП может считаться либо обязательным, либо нет.

Для того чтобы запретить сохранение неподписанных виз, установите флажок Подписывать резолюции электронной подписью при создании процесса.

Что такое процесс согласования

Процесс обходит всех участников согласования по заранее составленному списку с выбранным порядком - Все сразу, По очереди или Смешано. Для каждого исполнителя (роли) можно установить персональные сроки согласования.

Исполнять задачи можно по электронной почте, не заходя в программу (для этого должна быть включена персональная настройка Использовать выполнение задач по почте).

Результатов согласования может быть три: Согласован, Не согласован, Согласован с замечаниями. Если документ не согласован, можно отправить его на повторное согласование. Если согласован, удобнее всего через меню Создать на основании отправить далее - на утверждение, регистрацию, письмом электронной почты, создать мероприятие и т.п. В любой момент можно распечатать лист согласования с комментариями.
