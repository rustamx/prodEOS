При вызове из задачи "Согласовать" в данной форме  представлены результаты согласования всех исполнителей в текущем цикле и в прошлых циклах согласования.
Если форма открыта из задачи текущего цикла соглавания, то исполнитель может добавить нового сотрудника в список согласования.
Для этого предназначена кнопка «Добавить согласующего». При необходимости можно установить нового согласующего в нужную позицию в порядке согласования.
При этом в случае параллельного согласования автоматически будет создана задача для этого исполнителя.
Удалять и изменять согласующих, заданных автором процесса,  исполнитель не может.  
Если процесс создан из шаблона, для которого запрещено изменение процесса, то добавление нового согласующего запрещено.

При вызове из задачи "Ознакомиться с согласованием" в данной форме  представлены результаты согласования всех исполнителей в прошлых циклах согласования.
