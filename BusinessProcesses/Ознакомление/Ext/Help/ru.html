Ознакомление для нескольких сотрудников

Рекомендуем настроить шаблон "Ознакомление всех сотрудников" для того, чтобы каждый раз не добавлять большое количество сотрудников вручную.  

Чтобы подобрать нескольких сотрудников или целое подразделение, нажмите Подобрать. Добавить нескольких сотрудников можно только в не стартованный процесс.

Если процесс уже стартован, выбирайте сотрудников по одному кнопкой Добавить.

Как управлять запуском процесса

Для процесса можно настроить:

  • Отложенный старт - запуск процесса через определенный промежуток времени. Например, сразу после заключения договора создать процесс "Проверить отгрузку по договору" и отложить его запуск на месяц. Инструкция по настройке отложенного старта.
  • Автозапуск - запуск процесса после определенного события (настраивается администратором). Например, можно сделать так, чтобы зарегистрированный внутренний регламент сразу направлялся на ознакомления всем сотрудникам. Инструкция по настройке автозапуска.
  • Повторение - запускает процесс по указанному вами расписанию. Например, ознакомление с планом работ в последнюю пятницу каждого месяца. Инструкция по настройке повторения.

Как контролировать процесс

Процесс можно поставить на контроль. При этом будет создана контрольная карточка с записями о том, что контролировать, кого контролировать и на какой срок.

Есть несколько способов работы с контролируемыми процессами:

  • Следить за всеми контрольными карточками в окне На контроле;
  • Контролировать сроки с помощью записей календаря;
  • Получать на электронную почту уведомления об истекающих и приближающихся контрольных сроках.

Контрольную карточку можно снять с контроля , вернуть на контроль , изменить срок контроля , пометить на удаление.

Список ваших контрольных карточек не виден остальным сотрудникам (просмотр разрешен только вам, вашим руководителям и делегатам). Это позволяет использовать любые удобные для вас комментарии к контролируемым процессам.

Если настройка программы Использовать контроль объектов отключена, просто назначьте Контролера при создании процесса.

Что такое процесс ознакомления

С помощью данного бизнес-процесса документ, файл, письмо и т.п. рассылается всем пользователям (или ролям) по списку для ознакомления. К примеру, "Ознакомление сотрудников предприятия с новыми правилами безопасности".

Обратите внимание, что процесс завершится только после того, как все сотрудники подтвердят факт ознакомления с документом. Это очень важно, когда та или иная информация должна быть в обязательном порядке доведена до сведения сотрудников.
