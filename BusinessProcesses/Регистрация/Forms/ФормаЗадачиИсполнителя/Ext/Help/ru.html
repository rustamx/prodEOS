Как выполнить задачу регистрации

  1. Проверьте, утвержден ли документ. Зарегистрировать можно только утвержденный документ.
  2. Примите одно из двух решений: регистрировать или не регистрировать.

Для регистрации документа присвойте ему регистрационный номер (кнопка № в карточке документа). В зависимости от регламента вашего предприятия, добавьте  штрихкод или отправьте документ получателю и т.п. После этого нажмите Зарегистрировано.

Кнопка Не зарегистрирован служит для возврата документа на доработку. Отказаться от регистрации можно только перед присвоением регистрационного номера.

Если задача пришла к вам повторно, то по ссылке История регистрации можно посмотреть информацию о предыдущих циклах регистрации.

Как задать вопрос по задаче

В процессе работы над задачей вы можете задать вопрос коллеге. Для этого:

  1. Перейдите по ссылке Задать вопрос (если по задаче уже задан хотя бы один вопрос, ссылка будет иметь вид - Вопросы по задаче).   
  2. При необходимости, прикрепите дополнительные файлы и установите срок ответа.

Если ссылка недоступна, проверьте, установлена ли настройка программы Использовать рассмотрение вопросов выполнения задач.

Вы можете остановить или прервать отправленный вопрос соответствующими командами в командной панели его карточки.

Как перенести срок выполнения задачи

Если по каким-то причинам вы не успеваете выполнить задачу вовремя, отправьте автору запрос на перенос срока. Для этого:

  1. Перейдите по гиперссылке Срок либо выполните команду Согласовать перенос срока подменю Все действия.
  2. В карточке нового запроса на перенос срока введите желаемую дату и объясните причины переноса. При необходимости укажите срок ответа и прикрепите дополнительные файлы.

Если ни один из способов создания нового запроса на перенос срока не доступен, то либо автор не ограничил срок выполнения задачи, либо не установлена настройка программы Использовать рассмотрение вопросов выполнения задач.
