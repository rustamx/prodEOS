Как выполнить задачу контролера

Вы назначены контролером поручения, которое только что запущено.

В обязанности контролера обычно входит отслеживание сроков исполнения и соблюдение других регламентов предприятия. 

Вы можете следить за ходом и исполнением действий по процессу прямо из этой карточки. Для этого в карточке задачи отображается список Ход выполнения с установленным сроком, датой выполнения и комментариями.

Если вы видите, что исполнитель не успевает выполнить задачу в срок, отправьте автору запрос на перенос срока. Это можно сделать из карточки задачи исполнителя. Карточка открывается двойным кликом по строке задачи в списке Ход выполнения.

При завершении всех других задач по процессу, задача Контролировать закроется автоматически.

В программе есть возможность ставить на контроль любые объекты программы, следить за сроками и ходом их выполнения в одном окне. Для этого в настройках программы нужно включить настройку Использовать контроль объектов.

Как задать вопрос по задаче

Вопрос по задаче можно задать любому сотруднику. Для этого:

  1. Перейдите по ссылке Задать вопрос (если по задаче уже задан хотя бы один вопрос, ссылка будет иметь вид - Вопросы по задаче). 

  2. При необходимости, прикрепите дополнительные файлы и установите срок ответа.  

Если ссылка недоступна, проверьте, включена ли настройка программы Использовать рассмотрение вопросов выполнения задач.

После того, как автор вынесет решение, вы, при необходимости, сможете направить вопрос на повторное рассмотрение. Рассмотрение отправленного вопроса можно остановить или прервать из его карточки.

Как перенести срок выполнения задачи

Если вы по каким-то причинам не успеваете выполнить задачу, отправьте автору процесса запрос на перенос срока:

  1. Перейдите по гиперссылке Срок либо выполните команду Согласовать перенос срока подменю Еще.
  2. В карточке запроса на перенос срока введите желаемую дату и объясните причины переноса. При необходимости, укажите срок ответа и прикрепите дополнительные файлы.

Если ни один из способов создания нового запроса не доступен, то либо автор не ограничил срок выполнения задачи, либо не установлена настройка программы Использовать рассмотрение вопросов выполнения задач.
