Как выполнить задачу проверяющего

Вы назначены проверющим. Исполнитель уже выполнил отправленное ему поручение, теперь вы должны проверить результат.

Вы можете принять результат выполнения (Завершить поручение) или вернуть задачу на доработку.

При отправке на доработку рекомендуем подробно и четко указывать свои замечания, чтобы избежать лишних циклов проверки. Решать вопросы выполнения задачи рекомендуется с помощью ссылки Задать вопрос, а не возвращения на доработку. 

Процесс Поручение будет считаться незавершенным до тех пор, пока вы не закроете эту задачу.

Как задать вопрос по задаче

Вопрос по задаче можно задать любому сотруднику. Для этого:

  1. Перейдите по ссылке Задать вопрос (если по задаче уже задан хотя бы один вопрос, ссылка будет иметь вид - Вопросы по задаче). 
  2. При необходимости, прикрепите дополнительные файлы и установите срок ответа.  

Если ссылка недоступна, проверьте, включена ли настройка программы Использовать рассмотрение вопросов выполнения задач.

После того, как автор ответит на вопрос, вы сможете направить вопрос на повторное рассмотрение. Рассмотрение отправленного вопроса можно остановить или прервать из его карточки.
