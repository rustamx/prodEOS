Как контролировать комплексный процесс

Вы назначены контролером комплексного процесса. 

В обязанности контролера обычно входит отслеживание сроков исполнения и соблюдение других регламентов предприятия. Вы можете следить за ходом и исполнением действий по процессу прямо из карточки задачи контролера. Для этого есть список действий по процессу с установленным сроком, порядком и датой выполнения.

  • Открыть карточки процессов можно одноименной командой прямо из этого списка.
  • Изменить можно только те действия, по которым еще не стартовали процессы. В этом случае будет доступна команда Изменить действие. Действия с запущенными процессами выделяются зеленым цветом.
  • Просмотреть историю выполнения комплексного процесса можно по команде Все задачи панели навигации.

При завершении всех действий по процессу, ваша задача Контролировать закроется автоматически.

В программе есть возможность ставить на контроль любые объекты программы, следить за сроками и ходом выполнения нескольких объектов в одном окне. Для этого в настройках программы нужно включить настройку Использовать контроль объектов.

Как задать вопрос по задаче

Вопрос по задаче можно задать любому сотруднику. Для этого:

  1. Перейдите по ссылке Задать вопрос (если по задаче уже задан хотя бы один вопрос, ссылка будет иметь вид - Вопросы по задаче). 

  2. При необходимости, прикрепите дополнительные файлы и установите срок ответа.  

Если ссылка недоступна, проверьте, включена ли настройка программы Использовать рассмотрение вопросов выполнения задач.

После того, как автор вынесет решение, вы, при необходимости, сможете направить вопрос на повторное рассмотрение. Рассмотрение отправленного вопроса можно остановить или прервать из его карточки.

Как перенести срок выполнения задачи

Если вы видите, что исполнители не успевают в срок, отправьте автору процесса запрос на перенос срока. Для этого:

  1. Перейдите по гиперссылке Срок либо выполните команду Согласовать перенос срока подменю Еще.
  2. В карточке нового запроса на перенос срока введите желаемую дату и объясните причины переноса. При необходимости укажите срок ответа и прикрепите дополнительные файлы.

Если ни один из способов создания нового запроса не доступен, то либо автор не ограничил срок выполнения задачи, либо не установлена настройка программы Использовать рассмотрение вопросов выполнения задач.
