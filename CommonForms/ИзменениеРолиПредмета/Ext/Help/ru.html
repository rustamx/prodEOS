Предметы процесса бывают основными, вспомогательными и заполняемыми.

Заполняемый предмет можно указывать не для всех процессов.

Состояние основного предмета изменяется при выполнении задач процесса.

Состояние вспомогательного предмета не меняется в ходе выполнения задач процесса. Вспомогательным предметом может служить любой доступный объект в программе или файл на диске.

Заполняемый предмет предусматривается для одной из задач процесса и вводится на этапе ее выполнения. Если для задачи предусмотрен заполняемый предмет, то ее нельзя выполнить, пока предмет не заполнен.

Для заполняемого предмета указывается основание для заполнения, и задача, в которой он вводится. В шаблонах процессов также можно указать шаблон, по которому он должен быть создан.
