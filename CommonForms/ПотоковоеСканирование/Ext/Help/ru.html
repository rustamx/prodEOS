Порядок подачи бумаги

Порядок подачи бумаги выбирается в зависимости от характеристик сканера:

Двусторонний - при работе с двусторонним сканером.

Односторонний - при работе с односторонним сканером, когда сканируемые листы содержат информацию с одной стороны.

Последующие настройки нужны для одностороннего сканера в том случае, если сканируемые листы заполнены с обеих сторон.

От этих настроек будет зависеть в каком порядке будут располагаться отсканированные изображения:

Односторонний (сканер не меняет порядок листов при сканировании)

Односторонний (сканер меняет порядок листов при сканировании)

Режим разделения по документам

Чтобы отсканированные изображения автоматически добавились к нужному документу, перед сканированием их помечают штрихкодом, соответствующим штрихкоду этого документа.

Штрихкод на каждом листе - Предварительно нанесите штрихкод на каждую сканируемую страницу.
При сканировании программа будет искать штрихкод на каждом изображении и добавлять это изображение к соответствующему документу. Листы без штрихкода выводятся в списке нераспознанных изображений.

Штрихкод на пустом листе - Предварительно разделите сканируемые страницы пустыми листами с нанесенным  штрихкодом.
При сканировании программа будет добавлять все изображения после пустого листа со штрихкодом к соответствующему документу.

Штрихкод на первом листе - Предварительно нанесите штрихкод на первый лист каждой отдельной серии сканируемых листов.
При сканировании  программа будет добавлять все изображения после листа со штрихкодом (включая этот лист) к соответствующему документу.

Положение штрихкода

Задает расположение штрихкода на листе.
Можно указать угол, в котором расположен штрихкод, или задать координаты области со штрихкодом при помощи варианта Точное положение.
Если выбрано Не задано, компонента сканирования ищет штрихкод по всему листу.

Распределение изображений
по файлам

Задает режим сохранения сканируемых изображений:

Разделять пустой страницей - Изображения, разделенные пустыми листами, сохраняются в отдельных файлах. Пустые листы нужно заранее разместить между страницами документа.   

Все изображения в один файл - Все отсканированные изображения сохраняются в одном файле.

Каждое изображение в один файл - Каждое отсканированное изображение сохраняется в отдельный файл.

Если выбран режим Разделять пустой страницей, и используется ABBYY Recognition Server , то поле Сценарий распознавания текста становится обязательным для заполнения.

Использовать для распознавания

Для распознавания штрихкода можно использовать встроенную компоненту 1С:Документооборот 8 или программу ABBYY Recognition Server.

ABBYY Recognition Server не входит в комплект поставки 1С:Документооборот 8 и приобретается отдельно.

Имя или адрес компьютера
(только для ABBYY Recognition Server)		 Задает IP адрес или сетевое имя компьютера, на котором установлен ABBYY Recognition Server.
Сценарий распознавания штрихкодов
(только для ABBYY Recognition Server)		 Указывает название сценария для распознавания штрихкода.
Сценарий заранее создается с помощью Консоли удаленного администрирования ABBYY Recognition Server.
В настройках сценария укажите Распознать только штрихкоды (закладка Обработка) и выходной формат Текстовый документ с кодировкой Уникод UTF-16 (закладка Экспорт).

Сценарий распознавания текста
(только для ABBYY Recognition Server)

 

 Указывает название сценария для распознавания текста.
Сценарий заранее создается с помощью Консоли удаленного администрирования ABBYY Recognition Server.
В настройках сценария укажите Распознать весь текст и нужный язык распознавания (закладка Обработка), а также выходной формат Документ в формате HTML с настройкой Сохранять рисунки (закладка Экспорт).
Шаг поиска штрихкодов
(только для компоненты распознавания 1С:Документооборота)		 Задает интервал сканирования изображения (в пикселях) . По умолчанию равен 10.
Если штрихкод узкий и расположен под сильным наклоном, укажите меньшую цифру (от 1 до 10).
Очищать историю сканирования, кроме последних ... сеансов сканирования Если флажок установлен, то результаты сканирования будут удаляться за исключением заданного числа последних сеансов.

Настройка загрузки из каталога

Настройка Описание
Каталог Указывает каталог, откуда должны загружаться изображения (например, отсканированные сторонней программой)

Для сохранения в PDF нужно включить настройку Использовать ImageMagick для преобразования в PDF и установить программу ImageMagick.

Результаты сканирования штрихкодов

По завершении сканирования программа показывает число изображений:

  • Успешно размещенных,  т. е. добавленных к соответствующим документам в информационной базе,
  • Неразмещенных.

К неразмещенным программа может отнести, например, изображения:

  • С нераспознанным штрихкодом,
  • Не содержащие штрихкода (например, если используется режим Штрихкод на каждой странице)
  • С распознанным штрихкодом, который не соответствует документам в информационной базе.

Неразмещенные изображения можно вручную добавить к нужным документам.
