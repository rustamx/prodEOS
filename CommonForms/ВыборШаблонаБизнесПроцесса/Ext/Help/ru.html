Как применить шаблон процесса

Шаблон можно применить автоматически, интерактивно или вручную:

  • Для автоматического и интерактивного запуска шаблон должен быть Назначен видам документов. Инструкция по настройке автозапуска процесса по шаблону.
  • Чтобы применить шаблон вручную, используйте команду Заполнить по шаблону (подменю Еще в карточке процесса).

По одному шаблону может одновременно выполняться несколько процессов. Запущенные процессы можно открыть по команде Процессы по шаблону из карточки шаблона.

Какие данные указываются в шаблонах

В шаблоне можно указать:

  • Наименование и описание создаваемого процесса;
  • Срок исполнения, период отложенного старта и важность;
  • В качестве исполнителя в шаблоне можно выбрать пользователя, роль или автоподстановку и указать вариант исполнения (всем сразу или по очереди);
  • Условие маршрутизации;
  • Организацию (если ведется учет по несокльким организациям);
  • Предмет с настройками видимости и заполнения.

В настройках шаблона можно запретить пользователям вносить изменения в процесс, созданный по шаблону. Например, запретить менять список исполнителей.

Что такое шаблон процесса

Шаблон процесса - это набор заранее настроенных параметров (исполнители, сроки, условия маршрутизации и др.), которыми автоматически заполняется новый процесс. Шаблон настраивается в соответствие с принятым на предприятии регламентом движения документов и позволяет избежать повторного ввода одинаковых параметров вручную.

Например, можно сделать так, чтобы после регистрации входящего документа вида Коммерческое предложение он автоматически направлялся на рассмотрение директору по развитию (автозапуск процессов по шаблону).

Список пользователей и ролей, которые могут пользоваться шаблоном задается в Рабочей группе шаблона. Если рабочая группа не заполнена, то шаблон доступен всем.
