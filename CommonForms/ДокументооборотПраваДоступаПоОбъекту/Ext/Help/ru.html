В этом окне можно:

  • Посмотреть, какие права рассчитаны для выбранного вами файла.
  • Обновить права для данного файла. Перерасчет выполняется немедленно, даже если права файла стояли в очереди отложенного обновления.
  • Получить детальную техническую информацию по настроенным правам из Дескриптора доступа. Информация предназначена для технического специалиста.

Обратите внимание: окно Права доступа служит только для просмотра настроек прав файла. Редактирование прав в данном окне не предусмотрено.
