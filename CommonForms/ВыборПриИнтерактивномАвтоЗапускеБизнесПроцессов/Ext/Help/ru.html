Карточка запуска процесса сигнализирует о том, что для документов, созданных по выбранному при регистрации шаблону уже есть настроенный процесс. Из карточки запуска можно:

  • Либо стартовать процесс, нажав на кнопку Запустить процесс
  • Либо отменить запуск процесса, нажав на кнопку Не запускать

Если для текущего вида документов создано несколько шаблонов запуска процесса, то в карточке можно увидеть доступные варианты и сделать выбор. Настройкой шаблонов автоматического запуска процессов занимается ответственный за нормативно-справочную информацию.
