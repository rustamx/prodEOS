Для создания документа:

  1. Выберите один из предложенных шаблонов. В списке отображаются шаблоны, предназначенные только для создаваемого типа документа (входящего. исходящего или внутреннего). Шаблоны сгруппированы по дате последнего использования: Недавние - пять последних шаблонов, которые вы использовали. Все - остальные шаблоны для выбранного типа документа, включая Пустой.
  2. Нажмите Создать. При этом из указанного шаблона в создаваемый документ скопируются реквизиты, файлы, список личного доступа и категории.  

Для отмены создания документа предусмотрена кнопка Отмена.

Для создания документа с незаполненными полями:

  1. Выберите шаблон Пустой.
  2. Нажмите кнопку Создать.

Для заполнения документа по шаблону:

  1. Выберите один из предложенных шаблонов. В списке отображаются шаблоны, предназначенные только для создаваемого типа документа (входящего. исходящего или внутреннего). Шаблоны сгруппированы по дате последнего использования: Недавние - пять последних шаблонов, которые вы использовали. Все - остальные шаблоны для выбранного типа документа, включая Пустой.
  2. Нажмите Заполнить по шаблону. При этом из указанного шаблона в создаваемый документ скопируются реквизиты, файлы, список личного доступа и категории.
