Укажите полные пути к каталогам на серверах 1С:Предприятия.

Если в кластере серверов 1С:Предприятия используются компьютеры под управлением и Microsoft Windows и Linux, то необходимо указать оба пути.

Для серверов Windows используется путь в формате UNC (Uniform Naming Conventon), например:

\\servername\resource

Пример пути для серверов Linux:

\home\shared
