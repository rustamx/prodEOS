Как установить отбор

Отобрать нужные процессы можно двумя способами: с помощью полей быстрого отбора или с помощью команд подменю управления поиском.

  • Для быстрого поиска процессов в списке используйте поля отбора: по типу процесса, его состоянию, исполнителю или проекту.

Как контролировать процессы

Процесс можно поставить на контроль. При этом будет создана контрольная карточка с записями о том, что контролировать, кого и на какой срок.

Есть несколько способов работы с контролируемыми процессами:

  • Следить за всеми контрольными карточками в окне На контроле;
  • Контролировать сроки с помощью записей календаря;
  • Получать на электронную почту автоматические уведомления программы об истекающих и приближающихся контрольных сроках.

Как остановить и продолжить процесс

Для остановки процесса нажмите на командной панели Остановить, для возобновления работы - Продолжить.

Вместе с процессом остановятся все его незавершенные задачи и подпроцессы. Незавершенные задачи исчезнут из списков задач исполнителей, даже если они уже приняты к исполнению. При возобновлении работ процесс продолжится с момента, на котором был остановлен.

Для того чтобы остановить процесс безвозвратно, в карточке процесса предусмотрена команда Прервать.

Как настроить повторение процесса

Создание и старт процессов можно повторять по расписанию:

  1. Выберите процесс, который будет использован в качестве образца.
  2. Выполните команду Настроить повторение подменю Еще, откроется расписание повторения процесса.
  3. Настройте расписание повторения.
  4. Нажмите ОК, чтобы сохранить настройку.
