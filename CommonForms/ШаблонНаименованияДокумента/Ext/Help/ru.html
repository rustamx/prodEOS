Нумератор определяет правила формирования регистрационного номера разных видов документов. Вы можете настроить как последовательную нумерацию для всех документов, так и независимую по каждому в отдельности. Контроль присвоения регистрационного номера выполняется с учетом всех документов, для которых назначен этот нумератор.

Как привязать нумератор к виду документа

Чтобы нумератор определял формирование регистрационного номера для конкретного вида документа, свяжите их друг с другом:

  1. Перейдите в раздел Нормативно-справочная информация.
  2. Выполните команду Виды документов панели навигации.
  3. Откройте карточку вида документа из списка.
  4. На закладке Основные установите режим Автоматически, станет активным поле выбора нумератора.
  5. Выберите нумератор из списка.
  6. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.

Как настроить независимую нумерацию по связанному документу

Чтобы настроить независимую нумерацию по связанному документу, выполните следующие действия:

  1. Установите флажок Независимая нумерация, станет активным поле Тип связи.
  2. Выберите тип связи из списка или введите с клавиатуры вручную.
  3. Введите в поле Формат номера текст [НомерСвязДок].
  4. Нажмите на кнопку Записать и закрыть, чтобы сохранить настройку.

После настройки независимой нумерации убедитесь, что между видами документов установлен тот тип связи, который вы выбрали. Это можно сделать в списке Настройка связей (команда Настройка связей раздела Нормативно-справочная информация).

Если установлен флажок Независимая нумерация, но связанный документ не задан, при регистрации в карточке документа будет выдано сообщение об ошибке, а номер документа не будет сформирован.
