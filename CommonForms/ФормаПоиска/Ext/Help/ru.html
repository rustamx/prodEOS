Советы

Для простых случаев поиска достаточно ввести одно или несколько слов. Например: "Договор Цветметторг".

Для точного поиска используйте:

  • кавычки - для поиска точной фразы. Например: "О задержке сроков";
  • ИЛИ - для поиска любого из указанных слов. Например: "Клауст ИЛИ Энергомаш";
  • скобки - для группировки выражений. Например: "Петров (Клауст ИЛИ Энергомаш)".

Если полнотекстовый поиск ищет данные больше 5 секунд, скорее всего он долго готовит текст результатов поиска для отображения.

Чтобы решить подобную проблему:

  1. Перейдите по гиперссылке Расширенный поиск.
  2. Снимите флажок Отображать текстовое описание.
  3. Повторите поиск.

Что делать, если нашлось слишком много данных

Если программа нашла слишком много данных, конкретизируйте поиск. Для этого есть несколько способов:

  • Способ 1 - уточните запрос, добавьте к нему дату, сумму, название корреспондента и т.д. Например, если вы искали "Договор Клауст" и договоров с этим контрагентом нашлось слишком много, то укажите какое нибудь дополнительную подробность про это договор например слово ("договор клауст реконструкция") или дату ("договор клауст 2012") или сумму "договора клауст 120000".
  • Способ 2 -включите Расширенный поиск и укажите один или несколько разделов, в которых производить поиск.

Что делать, если ничего не нашлось

Если программа ничего не нашла:

  1. Упростите поисковое выражение, удалив одно или несколько слов. Не используйте кавычки. Включите дополнительно настройку Искать слова с ошибками.
  2. Если Вы использовали расширенный поиск, верните настройки поиска по умолчанию, для этого используйте кнопку .

Как добавить новый раздел поиска

Вы можете ограничить область поиска разделами, например, чтобы программа искала только среди внутренних документов или входящих писем. Вы можете использовать стандартные разделы или добавить свои собственные.

Чтобы создать новый раздел поиска, выполните следующие действия:

  1. Откройте контекстное меню списка разделов (правый щелчок мыши по списку).
  2. Выполните команду Настроить разделы поиска.
  3. В помощнике настройки разделов поиска, нажмите на кнопку Добавить.
  4. Дайте разделу понятное название, для этого заполните поле Наименование.
  5. Установите флажок у тех видов данных, которые Вы хотите включить в поиск.
  6. Нажмите ОК, чтобы сохранить настройку

Общие сведения

Полнотекстовый поиск в данных ищет и по реквизитам, и по содержимому файлов распространенных форматов (по умолчанию поддерживаются следующие форматы: ASCX, ASP, ASPX, CSS, HHC, HTA, HTM, HTML, HHT, HTW, HTX, ODC, STM, DOC, DOT, POT, PPS, PPT, XLB, XLC, XLS, XLT, TXT, EML; этот список также может быть расширен путем установки плагинов iFilter).
