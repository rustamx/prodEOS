Выберите один из шаблонов для создания процесса. Программа сгруппировала их в порядке, рекомендованном к использованию:

  • Рекомендованные шаблоны – шаблоны процессов, привязанные к выбранному вами виду документа (список выводится только при создании процесса на основании документа).
  • Общие шаблоны – шаблоны процессов, которые пока не назначены ни одному виду документов.
  • Новые процессы – список для создания процессов без шаблонов. Отображаются только те виды процессов, которые подходят к контексту вашей работы. Например, процесс Регистрация нельзя стартовать на основании контрагента или файла, поэтому в списке Новые процессы он представлен не будет.
  • Другие шаблоны – шаблоны процессов, которые настраивает ответственный за нормативно-справочную информацию. Если процесс создается из панели навигации раздела Совместная работа, в список попадут шаблоны, которые относятся хотя бы к одному из видов документов.

Чтобы открыть карточку шаблона процесса используйте клавишу F2.
