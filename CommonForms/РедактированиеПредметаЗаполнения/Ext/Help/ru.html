Заполняемый предмет предусматривается для одной из задач процесса и вводится на этапе ее выполнения. Если у задачи есть заполняемый предмет, то ее нельзя выполнить, пока предмет не заполнен.

Для заполняемого предмета указываются:

Имя предмета – уникальное для процесса имя, которое используется для отображения предмета исполнителям и передачи предмета между подпроцессами комплексного процесса;

Тип предмета – любой предмет, доступный для этого типа процесса;

Шаблон заполнения (только для шаблона процесса) для предмета в рамках выполнения соответствующей задачи;

На основании – предмет процесса или шаблона процесса, на основании которого будет введен заполняемый предмет;

В задаче – этап процесса, на котором должен быть заполнен предмет.
