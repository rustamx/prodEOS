Укажите имя предмета и его тип.

В качестве имени нужно задать любое понятное название предмета. Имя предмета используется для отображения предмета исполнителям и передачи предмета между подпроцессами комплексного процесса.

Тип предмета нуже для упорядочивания предметов. Список доступных типов предмета отличается для разных процессов.
