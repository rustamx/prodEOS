Отчет содержит несколько вариантов:

Список заключенных договоров - выводит список договоров, заключенных (зарегистрированных) за указанный период. Можно установить отбор на показ действующих (срок действия не закончился), не действующих (срок действия закончился) либо всех договоров. В список выводятся только зарегистрированные договоры. По умолчанию строки сгруппированы по контрагенту. Выводится количество договоров по каждому контрагенту и общее количество.

Сопроводительные документы по договору - для каждого договора выводится список связанных с ним документов. Можно отобрать связанные документа по типу связи, например, вывести только акты или накладные.  Для договора выводится сумма и валюта. Для сопроводительных документов выводится номер, дата и сумма, выраженная в валюте договора исходя из курса. Можно проанализировать полноту сопроводительных документов, сравнивая сумму договора с итоговой суммой сопроводительных документов. В отчет попадают только зарегистрированные сопроводительные документы.

Договоры с истекающим сроком действия - выводит список договоров, срок действия которых истекает в ближайшее время. Можно указать вывод документов, срок которых истекает на этой неделе, на следующей неделе, в этом месяце, в следующем месяце.

Расторгнутые договоры - выводит список расторгнутых договоров за указанный период.

Динамика количества заключенных договоров - формирует график изменения количества заключенных договоров по месяцам.

Динамика сумм заключенных договоров - формирует график изменения сумм заключенных договоров по месяцам. Суммы договоров пересчитываются в валюту, указанную в отчете.
