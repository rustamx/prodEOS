Отчет вызывается командой панели действий "Затраты времени" раздела "Учет рабочего времени".

Отчет предназначен для показа общих трудозатрат по виду работ в указанном периоде.
В поставку включены следующие варианты отчета:

  • Мои затраты времени - выводятся трудозатраты текущего пользователя в указанном периоде;
  • По видам работ - выводятся трудозатраты в указанном периоде, сгруппированные по видам работ и по подразделениям;
  • По подразделениям - выводятся трудозатраты в указанном периоде, сгруппированные по подразделениям, пользователям и по видам работ;
  • Статистика по видам работ - выводятся трудозатраты в указанном периоде в виде диаграммы, а также полный список трудозатрат по видам работ.

Управление текущими отборами осуществляется с помощью пользовательских настроек (список доступных (быстрых) настроек размещен ниже кнопки "Сформировать"). Полный список настроек доступен при выполнении команды "Настройка...".

Для формирования отчета укажите значение параметров "Период" и "Вид работ" (если требуется) и нажмите кнопку "Сформировать".
