Отчет "Статистика работ над документами" предназначен для выявления проблем в работе с документами. В данном отчете отражены завершенные процессы по документам. С помощью данного отчета можно узнать:

  • какие имеются проблемные документы в организации;
  • какие процессы происходят при работе над документами, сколько они занимают времени;
  • что является причиной медленной работы над документами.

Для работы с данным отчетом разработаны 5 вариантов. Один вариант является основным - "Статистика работ над документами", остальные варианты предназначены для расшифровок данного отчета.

Статистика работ над документами

Данный вариант отражает длительность работ по видам документов в разрезе различных типов процессов. В данном варианте отчета предусмотрены две основные расшифровки:

  • Количество документов - открывает вариант отчета "Документы по ответственным".
  • Средняя длительность - открывает вариант отчета "Средняя длительность процессов по документам".

Средняя длительность процессов по документам

Данный вариант открывается как расшифровка для варианта "Статистика работ над документами". Отражает длительность работ различным шаблонам (и без шаблонов). В данном варианте отчета предусмотрены три основные расшифровки:

  • Количество документов - открывает вариант отчета "Документы по ответственным".
  • Количество процессов, средняя длительность (по шаблону) - открывает вариант отчета "Процессы по шаблонам".
  • Количество процессов, средняя длительность (без шаблона) - открывает вариант отчета "Процессы по авторам".

Документы по ответственным

Данный вариант открывается как расшифровка для вариантов "Статистика работ над документами" и "Средняя длительность процессов по документам". Отражает длительность работ по различным документам для ответственных лиц. В данном варианте отчета предусмотрена специальная расшифровка:

  • Средняя длительность - открывает вариант отчета "Процессы по авторам".

Процессы по шаблонам

Данный вариант открывается как расшифровка для варианта "Средняя длительность процессов по документам". Отражает длительность различных процессов для различных шаблонов процессов. В данном варианте отчета нет специальных расшифровок.

Процессы по авторам

Данный вариант открывается как расшифровка для вариантов "Средняя длительность процессов по документам" и "Документы по ответственным". Отражает длительность различных процессов для различных авторов процессов. В данном варианте отчета нет специальных расшифровок.
