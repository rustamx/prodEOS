Отчет предназначен для контроля выполнения задач сотрудниками-исполнителями.

Для открытия формы отчета в разделе "Совместная работа" в панели действий "Отчеты" выполните команду "Справка об исполнительной дисциплине".

Для формирования отчета нажмите кнопку "Сформировать". По умолчанию в отчет попадут все задачи за текущий месяц.

Для детальной настройки отчета используйте команду "Все действия - Изменить вариант".

В отчете показываются задачи отчетного периода по исполнителям, разделенные на две условные категории: выполненные и не выполненные.

В колонке "Количество" показывается общее число задач.

По выполненным задачам: 
в колонке "Всего" показывается число всех выполненных задач, а в колонке "В срок" - число задач, выполненных в срок (реальный срок исполнения до планового), в колонке "Не в срок" - задачи, не выполненные в срок (реальный срок исполнения после планового).

По невыполненным задачам: 
в колонке "Всего" показывается число всех невыполненных задач, а в колонке "Просрочено" - число задач, срок исполнения которых истек.

Для настройки отчета используйте пользовательские настройки (периода, автора). Если требуется часто использовать определенные настройки отчета, рекомендуется создать для этих настроек вариант. Для использования варианта выберите его с помощью кнопки "Выбрать вариант".
