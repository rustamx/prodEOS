Отчет вызывается командой панели действий "Документооборот по организациям" раздела "Документы и файлы".

Отчет предназначен для показа документооборота по организациям, для которых в программе "1С:Документооборот" ведется учет, за указанный период.
В поставку включены следующие варианты отчета:

  • Внутренние документы - выводится количество внутренних документов. Данные списка дополняются графической информацией в виде диаграммы;
  • Входящие документы - выводится количество входящих документов. Данные списка дополняются графической информацией в виде диаграммы;
  • Задачи исполнителей - выводится число всех задач по организациям за указанный период. Отдельно выводится информация по выполненным задачам (в том числе в срок и не в срок) и не выполненным (в том числе просроченных);
  • Исходящие документы - выводится количество исходящих документов по каждой организации. Данные списка дополняются графической информацией в виде диаграммы.

Управление текущими отборами осуществляется с помощью пользовательских настроек (список доступных (быстрых) настроек размещен ниже кнопки "Сформировать"). Полный список настроек доступен при выполнении команды "Настройка...".

Для построения отчета необходимо выбрать нужный вариант, установить использование параметров отбора и указать значения отбора, нажать кнопку "Сформировать".

Настройка каждого варианта может быть изменена, а также созданы другие варианты отчета.
