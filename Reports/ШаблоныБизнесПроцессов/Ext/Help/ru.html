Отчет вызывается командой панели действий "Шаблоны процессов" раздела "Совместная работа".

Отчет предназначен для показа использования шаблонов процессов в указанном периоде.
В поставку включены следующие варианты отчета:

  • Количество процессов по шаблонам - выводится статистический список использования шаблонов. Данные списка дополняются графической информацией в виде диаграммы;
  • Список процессов по шаблонам - выводится список процессов, сгруппированных по видам используемых в указанном периоде шаблонов;
  • Список неиспользуемых шаблонов - выводится список неиспользуемых в указанном периоде шаблонов;
  • Статистика использования шаблонов - выводится диаграмма, показывающая использование шаблонов для каждого вида процесса. Цветом показываются данные по процессам, созданных с использование и без использования шаблонов.

Управление текущими отборами осуществляется с помощью пользовательских настроек (список доступных (быстрых) настроек размещен ниже кнопки "Сформировать"). Полный список настроек доступен при выполнении команды "Настройка...".

Для построения отчета необходимо выбрать нужный вариант, установить использование параметров отбора и указать значения отбора, нажать кнопку "Сформировать".

Настройка каждого варианта может быть изменена, а также созданы другие варианты отчета.
